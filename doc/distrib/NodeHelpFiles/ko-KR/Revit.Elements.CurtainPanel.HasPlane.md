## 상세
`CurtainPanel.HasPlane`은 입력 커튼 패널이 평면인지 여부를 확인합니다. 평면이 아니면 false 값이 반환됩니다. 평면인 경우 true 값이 반환됩니다.

아래 예에서는 커튼월 요소가 벽 요소 내의 모든 커튼 패널과 함께 요소 ID로 선택됩니다. 그런 다음 커튼 패널이 평면인지 여부가 `CurtainPanel.HasPlane`을 사용하여 확인됩니다.
