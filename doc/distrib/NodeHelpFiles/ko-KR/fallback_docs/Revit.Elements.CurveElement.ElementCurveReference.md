## 상세
이 노드는 모델 곡선 또는 상세 선과 같은 지정된 Revit CurveElement와 연관된 곡선 참조를 검색합니다. 그런 다음 이 참조를 치수기입, 정렬 또는 분할된 경로 작성 등의 형상 참조가 필요한 다른 노드에 대한 입력으로 사용할 수 있습니다.

이 예에서는 시작점과 끝점을 사용하여 모델 곡선을 작성한 다음 CurveElement.ElementCurveReference 노드에 입력합니다. 출력은 다운스트림 작업에서 사용할 수 있는 곡선 요소의 기하학적 참조입니다.
