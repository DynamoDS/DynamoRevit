## 상세
`FamilyParameter.StorageType`은 지정된 매개변수가 저장하는 데이터 유형을 보고합니다.

가능한 옵션은 다음과 같습니다.
- Double
- 정수
- 문자열
- 요소 ID

아래 예에서는 현재 Revit 문서가 패밀리 문서로 변환되고 모든 매개변수를 가져옵니다. `FamilyParameter.StorageType`은 패밀리에 있는 모든 매개변수의 저장소 유형을 가져오는 데 사용됩니다.
