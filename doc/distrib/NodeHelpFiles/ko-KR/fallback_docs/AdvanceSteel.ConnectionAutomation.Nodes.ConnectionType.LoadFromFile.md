## 상세
이 노드는 외부 Revit(.rvt) 파일에서 로드할 수 있는 사용 가능한 연결 유형 리스트를 반환합니다.

이 예에서는 제공된 파일 경로에서 외부 Revit 파일에 연관된 Revit 모델에서 사용할 수 있는 연결 리스트를 반환합니다. 로드된 연결 유형이 모델에 아직 없어야 합니다. 그렇지 않으면 오류가 발생합니다.
