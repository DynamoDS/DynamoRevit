## 상세
`Mullion.AsFamilyInstance`는 입력 멀리언 요소를 Revit 패밀리 인스턴스로 변환합니다. 이는 요소 매개변수를 가져오거나 멀리언에서 유형 특성을 추출하는 데 유용합니다.

아래 예에서는 모든 커튼 패널을 요소 ID로 선택한 커튼월 요소에서 가져옵니다. 그런 다음 커튼월 패널에서 멀리언을 가져옵니다. 마지막으로 멀리언이 패밀리 인스턴스로 변환됩니다.
