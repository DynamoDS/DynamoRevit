## 상세
경계 상자 입력을 기반으로 단면 뷰를 작성합니다.

이 예제에서는 Revit에서 단면 뷰 작성을 자동화하는 방법을 보여줍니다. 먼저 두 개의 벽을 선택하고 해당 벽의 경계 상자를 생성합니다. 마지막으로, 벽의 경계 상자 크기에 맞게 정확히 조정된 두 개의 새로운 단면 뷰를 작성합니다. 이 노드는 형상의 정확한 치수를 기반으로 단면을 생성할 때 적합합니다.
