## 상세
이 노드는 제공된 면적 양식(건축 면적 또는 임대 가능) 및 파일에 대한 고유한 이름을 사용하여 면적 일람표를 생성합니다.

이 예제에서는 모든 면적 양식을 수집하고 첫 번째 면적 양식을 선택합니다. 그런 다음, 해당 이름을 'Dynamo Schedule'의 접두어로 사용하여 생성된 면적 일람표에 대한 고유한 이름을 작성합니다.
