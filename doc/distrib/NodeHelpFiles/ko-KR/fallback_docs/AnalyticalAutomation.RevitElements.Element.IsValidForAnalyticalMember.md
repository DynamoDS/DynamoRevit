## 상세
이 노드는 선택한 Revit 요소를 평가하여 해석 부재로 사용하기에 적합한지 결정합니다. 요소가 유효하지 않은 경우 이유에 대해 설명하는 예외 메시지가 출력에 포함됩니다.

이 예에서는 요소가 요소 ID로 정의되고 Element.IsValidForAnalyticalMember에 제공됩니다. 출력에는 요소의 유효성을 나타내는 부울 결과와 해당되는 경우 예외 메시지가 포함됩니다.
