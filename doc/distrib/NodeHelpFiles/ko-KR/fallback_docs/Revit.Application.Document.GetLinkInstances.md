## 상세
`Document.GetLinkInstances`는 현재 Revit 문서에 배치된 링크의 모든 인스턴스 리스트를 반환합니다. 링크 인스턴스는 링크된 파일에서 요소를 검색하는 데 사용됩니다.

아래 예제에서는 모든 Revit 링크 인스턴스가 활성 Revit 문서에 대해 반환됩니다. 또한 링크의 이름이 `Element.Name`을 사용하여 반환됩니다.
