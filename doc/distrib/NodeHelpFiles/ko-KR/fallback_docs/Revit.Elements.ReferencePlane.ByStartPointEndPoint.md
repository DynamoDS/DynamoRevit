## 상세
이 노드는 시작점과 끝점을 정의하여 모델에 새 참조 평면을 만듭니다.

이 예에서는 숫자 입력을 사용하는 Point.ByCoordinates로 두 개의 점이 생성됩니다. 그런 다음 이러한 점이 ReferencePlane.ByStartPointEndPoint로 피드되어 두 좌표 사이에 직접 사용자 참조 평면을 작성합니다.
