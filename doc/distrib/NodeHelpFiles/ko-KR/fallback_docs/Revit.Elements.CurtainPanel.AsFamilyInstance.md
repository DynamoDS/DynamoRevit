## 상세
`CurtainPanel.AsFamilyInstance`는 입력 커튼 패널 요소를 Revit 패밀리 인스턴스로 변환합니다. 이는 요소 매개변수를 가져오거나 커튼 패널에서 유형 특성을 추출하는 데 유용합니다.

아래 예에서는 요소 ID로 선택한 커튼월 요소에서 모든 커튼 패널을 가져옵니다. 그런 다음 커튼 패널이 패밀리 인스턴스로 변환됩니다.
