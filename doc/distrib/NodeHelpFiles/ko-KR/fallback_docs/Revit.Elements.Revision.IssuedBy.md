## 상세
Dynamo의 Revision.IssuedBy 노드는 Revit의 수정기호에서 "발행자" 정보를 읽는 데 사용됩니다. 이를 통해 팀은 수정기호를 작성하거나 발행한 사람을 신속하게 식별할 수 있으므로 문서 투명성이 향상되고 추적이 쉬워집니다.

이 그래프에서 수정기호 선택 노드는 프로젝트에서 수정기호를 선택하는 데 사용됩니다. Revision.IssuedBy 노드는 "발행자" 값을 추출하고 Watch 노드에 결과가 표시됩니다. 이 설정을 사용하면 사용자가 Revit의 수정기호 관리자를 수동으로 열지 않고도 수정기호 소유권을 검토할 수 있습니다.
