## 상세
이 노드는 선택한 단면 모양으로 필터링된 연결 노드 리스트를 반환합니다.

여러 연결 노드가 있는 일련의 구조 데이터 요소를 선택할 때 이 예에서는 연결 노드 리스트를 선택한 단면 모양과 일치하는 연결 노드로 필터링합니다. 구조 데이터 요소에는 항목의 하위 리스트가 있을 수 있으므로 색인이 기본값인 0으로 설정됩니다.

이 예에서는 I 모양 평행 플랜지 그룹과 HSS를 선택하고 출력은 "accepted" 및 "rejected" 특성을 가진 사전 목록으로 구성됩니다. accepted 값은 선택한 단면 모양으로 필터링된 연결 노드입니다. 이 경우 단면 모양은 'HSS'입니다.
