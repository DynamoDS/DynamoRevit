## 상세
이 노드는 구조 해석 결과 또는 Revit 해석 모델에서 지정된 해석 결과 유형(예: Fx, Fy, Fz, Mx, My, Mz)에 대한 하중 값을 검색합니다. 선택한 구조 요소의 각 연결 구역에서 힘을 평가합니다.

이 예에서는 선택한 모델 해석 모드를 기반으로 기둥을 선택하여 링크된 각 연결 구역에 Fz 힘 구성요소를 표시합니다.
