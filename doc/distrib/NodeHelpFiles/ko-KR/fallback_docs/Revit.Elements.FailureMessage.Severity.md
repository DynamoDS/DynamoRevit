## 상세
이 노드는 지정된 FailureMessage 객체의 심각도 레벨을 검색합니다. 심각도 값은 "경고", "오류" 및 "중요"입니다.

이 예에서는 "뷰 자르기가 사용되지 않도록 설정되었습니다." 조건을 확인하기 위해 성능 어드바이저 규칙이 실행됩니다. 생성된 실패 메시지는 각 문제의 심각도 레벨을 보고하는 FailureMessage.Severity 노드로 전달됩니다.
