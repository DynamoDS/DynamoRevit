## 상세
`FamilyParameter.CanAssignFormula`는 지정된 패밀리 매개변수에 수식을 할당할 수 있는지 여부를 나타내는 true 또는 false 값을 반환합니다.

아래 예에서는 현재 Revit 문서(샘플 모델의 문 패밀리)가 열립니다. 패밀리의 매개변수가 수식을 할당할 수 있는지 여부를 나타내는 부울 값 리스트와 함께 반환됩니다.
