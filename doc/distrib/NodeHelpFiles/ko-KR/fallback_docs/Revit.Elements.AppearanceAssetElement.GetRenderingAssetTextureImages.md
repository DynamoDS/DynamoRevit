## 상세
이 노드는 Revit에서 지정된 AppearanceAssetElement의 렌더링 자산과 연관된 모든 텍스처 이미지를 검색합니다. 모양 자산은 재료의 시각적 특성(예: 색상, 반사도, 범프 또는 투명)을 정의합니다.

이 예에서는 AppearanceAssetElement가 선택되어 AppearanceAssetElement.GetRenderingAssetTextureImages 노드에 대한 입력으로 사용됩니다. 결과는 지정된 모양 자산 요소에 대한 이미지 특성 및 이미지 경로입니다. 모양 자산 요소에 값이 없는 경우 빈 리스트가 표시됩니다.
