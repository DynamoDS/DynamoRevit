## 상세
이 노드는 가변 패밀리 유형의 배치 위치를 정의하고 선택한 면에 UV 매개변수 값을 적용하여 가변 구성요소를 배치합니다.

이 예에서는 곡선을 돌출(수동으로 수행)하여 매스 패밀리 내에 표면을 작성하고, 해당 표면이 면 입력으로 선택됩니다. 그런 다음 UV 값을 제공하여 배치 위치를 결정하고 Diagnostic Tripod – 1 Point.rfa 패밀리가 유형으로 사용됩니다. AdaptiveComponent.ByParametersOnFace 노드는 선택한 면에 배치된 가변 구성요소를 출력합니다. 이 그래프를 실행하기 전에 "Diagnostic Tripod – 1 Point.rfa"를 매스 패밀리에 로드해야 합니다.

이 노드 도움말 예제 파일을 실행하려면 "Diagnostics Tripod-1 point.rfa"를 Revit 파일로 로드해야 합니다. 패밀리가 여기에 저장됩니다. C:\ProgramData\Autodesk\RVT 2027\Dynamo\Samples\Data
