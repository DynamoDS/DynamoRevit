## 상세
이 노드는 성능 어드바이저 규칙에 의해 생성된 실패 또는 경고에 대한 자세한 설명을 검색합니다. 오류를 명확하고 읽기 쉬운 텍스트로 변환하여 사용자가 문제를 더 잘 이해할 수 있도록 합니다.

이 예에서는 성능 어드바이저 규칙은 "뷰 자르기가 사용되지 않도록 설정"되어 있는지 여부를 확인합니다. 결과는 실패의 원인을 설명하는 설명 메시지를 출력하는 FailureMessage.Description 노드를 통해 처리됩니다.
