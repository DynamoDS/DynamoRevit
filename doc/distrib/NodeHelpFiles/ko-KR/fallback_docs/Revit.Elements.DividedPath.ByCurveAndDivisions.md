## 상세
이 노드는 지정된 곡선을 지정된 동일한 세그먼트 수로 분할하여 매스작업 패밀리에 분할된 경로 요소를 작성합니다.

이 예에서는 모델 곡선이 지정된 시작점과 끝점에서 작성된 다음 DividedPath.ByCurveAndDivisions 노드의 입력 곡선으로 사용됩니다. 정수 값은 곡선을 따라 동일한 분할 수를 정의합니다. 마지막 노드에는 곡선을 따라 생성된 분할 점이 표시됩니다.
