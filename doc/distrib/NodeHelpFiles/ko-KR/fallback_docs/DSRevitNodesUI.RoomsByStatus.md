## 상세
`Rooms By Status`는 문서의 모든 룸을 상태를 기준으로 그룹화하여 반환합니다.

상태에는 다음과 같은 룸이 포함됩니다.
- 올바르게 배치되고 경계가 지정된 룸
- 배치되지 않은 룸(이전에 뷰에서는 삭제되었지만 모델에서는 삭제되지 않은 룸)
- 닫히지 않은 룸(계산된 면적이 없고 경계가 지정되지 않은 룸)
- 중복 룸(다른 룸과 닫힌 공간을 공유하는 룸)

아래 예에서는 배치된 룸의 면적이 조회됩니다.
