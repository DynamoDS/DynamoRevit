## 상세
이 노드는 곡선을 지정된 수의 동일한 세그먼트로 분할하여 매스작업 패밀리에 분할된 경로 요소를 작성합니다.

이 예에서는 지정된 시작점과 끝점에서 2개의 모델 곡선을 작성한 다음, DividedPath.ByCurvesAndDivisions 노드의 입력 곡선으로 사용합니다. 정수 값은 곡선을 따라 동일한 분할 수를 정의합니다. 마지막 노드에는 곡선을 따라 생성된 분할 점이 표시됩니다.
