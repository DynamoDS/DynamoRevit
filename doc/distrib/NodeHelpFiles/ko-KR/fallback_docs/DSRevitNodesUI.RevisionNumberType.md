## 상세
이 노드를 사용하면 Revit 프로젝트에서 사용 가능한 옵션 중에서 수정기호 번호 유형을 선택할 수 있습니다. 일반적으로 수정기호 시퀀스 내에서 수정기호에 번호나 문자가 지정되는 방법을 정의하는 데 사용됩니다(예: 숫자, 영숫자 또는 없음).

이 예에서는 수정기호 번호 유형 선택 노드의 값인 "영숫자"를 사용하여 새 수정기호를 작성합니다.

수정기호에 대한 자세한 내용은 링크를 참고하십시오.
https://help.autodesk.com/view/RVT/2025/KOR/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
