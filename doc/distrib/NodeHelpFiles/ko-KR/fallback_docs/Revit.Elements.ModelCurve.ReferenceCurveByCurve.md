## 상세
이 노드는 Revit 패밀리 문서 내에 Revit 참조선(참조 곡선)을 생성합니다.

이 예에서는 숫자 슬라이더로 생성된 좌표가 모델 곡선의 시작점과 끝점을 정의하며, 그런 다음 이 노드에 공급됩니다. 시퀀스의 마지막 노드는 새로 생성된 참조선에 대한 모든 매개변수 및 해당 값을 표시합니다. 이 노드의 기능은 Revit 패밀리 문서로 제한됩니다.
