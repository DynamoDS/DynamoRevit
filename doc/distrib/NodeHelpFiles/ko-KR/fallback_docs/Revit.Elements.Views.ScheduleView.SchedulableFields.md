## 상세
`ScheduleView.Fields`와 마찬가지로 `ScheduleView.SchedulableFields`는 일람표 뷰에 적용되는 필드를 검색합니다. 그러나 이러한 필드에는 사용 여부에 관계없이 지정된 일람표 뷰 유형에 사용 가능한 모든 매개변수가 포함됩니다.

아래 예에서는 현재 Revit 파일에서 문 일람표가 수집되고 일람표에 사용 가능한 필드가 보고됩니다.
