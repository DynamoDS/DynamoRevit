## 상세
이 노드는 사용 가능한 모든 성능 어드바이저 규칙의 리스트를 검색합니다. 이러한 규칙은 Revit에서 잠재적인 성능 또는 모델링 문제를 식별하는 데 사용하는 내장 검사와 동일합니다.

이 예에서는 PerformanceAdviserRules 노드를 사용하여 전체 리스트에서 "뷰 자르기가 사용되지 않도록 설정되었습니다." 규칙을 가져온 다음 다른 노드에 연결하여 해당 속성이나 동작을 확인할 수 있습니다. 이를 통해 사용자는 특정 성능 규칙으로 빠르게 작업할 수 있습니다.
