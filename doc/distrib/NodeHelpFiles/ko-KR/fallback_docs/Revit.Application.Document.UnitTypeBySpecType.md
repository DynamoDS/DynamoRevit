## 상세
이 노드는 지정된 사양 유형에 대해 프로젝트에서 사용 중인 단위를 가져옵니다.

이 예에서는 사양 유형 값(면적 및 면적당 비용)이 리스트에 입력됩니다. 이 리스트는 Document.UnitTypeBySpecType 노드에서 specType의 입력으로 사용되고 현재 문서는 문서 입력으로 사용됩니다. 결과에는 현재 프로젝트에 대한 프로젝트별 사양 유형 단위 값이 표시됩니다.
