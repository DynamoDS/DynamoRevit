## 상세
Revit 프로젝트 내의 특정 점에 Revit 패밀리 유형 인스턴스를 배치하며, 각 인스턴스를 지정된 레벨과 연계합니다.

이 예제에서는 패밀리 유형, 점, 레벨을 입력하여 패밀리 인스턴스를 작성합니다. 이 노드는 스프레드시트나 기타 생성된 데이터와 같은 소스의 데이터를 사용하여 Revit 내 가구, 설비, 장비 등의 요소 배치를 자동화할 때 활용할 수 있습니다.
