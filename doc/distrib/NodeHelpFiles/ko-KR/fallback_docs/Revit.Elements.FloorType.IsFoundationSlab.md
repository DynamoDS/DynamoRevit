## 상세
`FloorType.IsFoundationSlab`는 지정된 바닥 유형이 기초 슬래브인지 여부를 나타내는 true 또는 false 값을 반환합니다.

아래 예에서는 현재 Revit 문서에 있는 모든 바닥이 수집됩니다. 그런 다음 바닥이 유형으로 그룹화되고 `FloorType.IsFoundationSlab`를 통해 바닥 유형이 기초 슬래브인지 여부가 확인됩니다.
