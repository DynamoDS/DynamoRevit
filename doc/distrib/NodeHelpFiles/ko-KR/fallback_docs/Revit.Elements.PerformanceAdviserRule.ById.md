## 상세
이 노드는 고유한 규칙 ID를 사용하여 Revit에서 특정 성능 어드바이저 규칙을 검색합니다.

이 예에서는 "프로젝트에 사용되지 않은 패밀리 및 유형이 포함되어 있습니다." 규칙이 선택되고 해당 규칙 ID가 추출됩니다. 이 ID는 문자열로 변환된 다음 PerformanceAdviserRule.ById로 다시 전달되며, 이 PerformanceAdviserRule.ById는 향후 사용 또는 자동화를 위해 동일한 규칙을 다시 식별하고 검색합니다.
