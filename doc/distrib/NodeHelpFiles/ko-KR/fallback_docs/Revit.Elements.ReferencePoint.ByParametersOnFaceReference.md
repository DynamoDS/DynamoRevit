## 상세
`ReferencePoint.ByParametersOnFaceReference`는 활성 Revit 패밀리 문서에서 선택된 표면의 지정된 매개변수에 참조 점 요소를 작성합니다.

참고: 패밀리 문서는 가변 구성요소 또는 매스 패밀리여야 합니다.

아래 예제에서는 참조 점이 패밀리 문서에서 선택된 표면의 중심에 작성됩니다.
