## 상세
이 노드는 Revit 링크 인스턴스 내에서 지정된 레벨에 호스트되거나 연관된 모든 요소를 반환합니다.

이 예에서는 모든 Revit 링크를 선택하고 레벨과 함께 LinkInstance.AllElementsAtLevel에 입력합니다. 출력은 레벨 L3에 있거나 레벨 L3과 관련된 모든 Revit 링크 요소를 노출합니다. 마지막 두 노드는 링크 변환과 역 변환을 제공합니다.
