## 상세
이 노드는 활성 Revit 프로젝트에서 길이 측정에 사용되는 단위 유형을 식별하고 해당 단위 유형 식별자를 문자열 값으로 출력합니다.

이 예에서는 UnitsUtilities.GetCurrentProjectLengthUnit 노드가 해당 출력에 연결된 Watch 노드와 함께 그래프에 배치됩니다. Watch 노드에는 현재 프로젝트 길이 단위가 문자열(예: 피트)로 표시됩니다.
