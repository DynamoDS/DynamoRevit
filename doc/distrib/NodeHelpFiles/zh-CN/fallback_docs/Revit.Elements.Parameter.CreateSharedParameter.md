## 详细
“Parameter.CreateSharedParameter”会将具有给定值的参数添加到当前 Revit 文档中的给定类别。此节点与“Parameter.CreateSharedParameterForAllCategories”不同，允许输入特定类别。

在下面的示例中，名为“包含在 FFE 中”的共享“是/否”参数已添加到当前 Revit 文档中的家具和常规模型类别。
