## 详细
此节点返回按选定截面形状过滤的连接节点列表。

选择具有多个连接节点的一系列结构数据图元时，本例会将连接节点列表过滤为与选定剖面形状匹配的连接节点。结构数据图元可以包含项目的子列表，因此索引设置为 0(默认值)。

在该示例中，选择了一组 I 形平行翼缘和一个 HSS，输出由具有“已接受”和“已拒绝”特性的词典列表组成。接受的值是按选定剖面形状过滤的连接节点。在这种情况下，剖面形状为“HSS”。
