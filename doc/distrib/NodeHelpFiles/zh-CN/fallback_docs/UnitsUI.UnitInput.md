## 详细
`Parse Unit Input` accepts a numeric value together with a unit type, providing a way to define a quantity such as length, area, volume, angle, or another measurable value in a controlled format.

In the example below, `Parse Unit Input` is used with a value of 10 and the unit Feet. The outputs are a `System.Double` value of 10 and the unit name Feet.
