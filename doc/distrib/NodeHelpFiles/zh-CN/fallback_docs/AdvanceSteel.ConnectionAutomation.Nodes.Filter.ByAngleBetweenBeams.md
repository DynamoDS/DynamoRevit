## 详细
此节点返回按最小-最大度数范围过滤的连接节点列表。

在此例中，选择了 W 形柱和有角度的 HSS 支撑，输出由具有“已接受”和“已拒绝”特性的词典列表组成。接受的值是处于 0 到 60 度范围内的连接节点(此节点从 X 轴测量)。如果最大值为 45 度，则连接节点将被拒绝。“indexFirst”和“indexSecond”参数设置为“使用标高”，以与输入数据结构保持一致。
