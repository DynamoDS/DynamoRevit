## 详细
此节点返回按选定截面轮廓过滤的连接节点列表。

选择具有多个连接节点的一系列结构数据图元时，会将连接节点列表过滤为选定的剖面轮廓。结构数据图元可以包含项目的子列表，因此索引设置为 0(默认值)。

在该示例中，选择了一组 W 形，输出由具有“已接受”和“已拒绝”特性的词典列表组成。值“已接受”是按选定截面轮廓过滤的连接节点。在这种情况下，剖面轮廓为“W 形柱”。
