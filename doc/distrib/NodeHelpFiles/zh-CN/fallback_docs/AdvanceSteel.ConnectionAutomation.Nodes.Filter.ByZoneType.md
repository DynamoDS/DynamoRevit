## 详细

此节点返回按选定分区类型(端点或主体)过滤的连接节点列表。

当选择一系列具有多个连接节点的结构数据图元时，本例将过滤具有选定剖面轮廓的连接节点列表。结构数据图元可以包含项目的子列表，因此索引设置为 0(默认值)。

在该示例中，选择了一组 I 形平行凸缘，输出由具有“已接受”和“已拒绝”特性的词典列表组成。接受的值是按选定分区类型过滤的连接节点。在这种情况下，分区类型为“端点”。
