## 详细
根据 sheetName、sheetNumber、titleBlockFamilyType、Views 及其位置的输入创建图纸。

在本例中，将使用 Point.ByCoordinate 输入创建一个名为 DynamoDemo10 的图纸，其中包含两个独立定位的视图。输入视图不能位于现有图纸上。此节点的目的是在 Revit 中创建图纸，同时在定义的位置处放置指定的视图。
