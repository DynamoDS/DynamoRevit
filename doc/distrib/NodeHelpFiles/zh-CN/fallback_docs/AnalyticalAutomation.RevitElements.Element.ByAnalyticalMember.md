## 详细
此节点从分析构件创建物理结构图元，或更新已连接到分析构件的现有物理图元。该节点可以根据分析构件更新物理图元的几何图形或参数，具体取决于输入设置。当分析构件不与任何物理图元相关联时，将使用分析构件的信息创建物理图元。

在此示例中，从结构柱中选择了 ACO Supermarket 项目中的分析构件。该分析构件连接到此节点，并且布尔输入用于显示用于更新几何图形和参数的默认设置。该节点根据分析模型生成相应的物理柱。
