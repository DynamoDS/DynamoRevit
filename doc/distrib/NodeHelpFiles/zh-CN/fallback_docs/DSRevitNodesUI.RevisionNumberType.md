## 详细
此节点用于从 Revit 项目的可用选项中选择“修订编号类型”。它通常用于定义修订在修订序列中如何对修订进行编号或添加字母(例如数字、字母数字或无)。

在此示例中，使用“Select Revision Number Type”(选择修订号类型)节点创建新修订，值为“Alphanumeric”。

有关修订的详细信息，请参见链接。
https://help.autodesk.com/view/RVT/2025/CHS/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
