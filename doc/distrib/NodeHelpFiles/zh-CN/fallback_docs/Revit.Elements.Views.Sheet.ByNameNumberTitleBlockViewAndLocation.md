## 详细
根据 sheetName、sheetNumber、titleBlockFamilyType、View 的输入及其位置创建图纸。

在本例中，将使用 Point.ByCoordinate 输入创建一个名为 DynamoDemo11 的图纸，其中包含一个独立定位的视图。输入视图不能位于现有图纸上。此节点的目的是在 Revit 中创建图纸，同时在定义的位置处放置指定的视图。
