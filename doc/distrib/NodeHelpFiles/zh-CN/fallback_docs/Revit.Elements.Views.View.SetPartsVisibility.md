## 详细
此节点设置零件在指定 Revit 视图中的显示方式。此节点允许您控制视图是否显示；“ShowPartsOnly”、“ShowOriginalOnly”、“ShowPartsAndOriginal”或“unset”。

在此示例中，将选择视图(三维视图)并用作节点 View.SetPartsVisibility 的输入(视图)以及节点“View Parts Visibilities”中的任何默认值。最后 2 个节点显示原始设置及其修改后的设置的视图零件可见性设置。
