## 详细
此节点返回应用于 Revit 中给定分割表面的旋转角度(以度为单位)。旋转定义曲面上的栅格图案如何相对于曲面的自然 U 和 V 方向进行定向。

在此示例中，将选择一个面，并将其用于创建分割表面。此分割曲面输出用作 DividedSurface.Rotation 节点的输入。输出是应用于分割表面的旋转因子(以度为单位)。最后的节点会显示分割表面的值。运行此示例图形时，您需要遵照 Revit 警告并删除建议的图元，以便轴网显示在选定表面上。

有关更多信息，请参见链接。
