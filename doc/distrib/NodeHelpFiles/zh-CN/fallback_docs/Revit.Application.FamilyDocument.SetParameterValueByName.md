## 详细
“FamilyDocument.SetParameterValueByName”将给定参数设置为族文档中选定族类型的给定值。

在下面的示例中，当前文件将转换为族文档。然后，将参数“半径”的值设置为“36" 直径”族类型的“数字滑块”值。
