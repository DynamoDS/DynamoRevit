## 详细
“Coordinates.BasePoint”返回项目基点作为点几何图形。项目基点是在 Revit 中设置的，并且可能因项目而异。有关 Revit 中项目基点的详细信息，请访问此 [帮助文章](https://help.autodesk.com/view/RVT/2025/CHS/?guid=GUID-30D76259-CC67-4498-B06B-91F7517F9B65)。
