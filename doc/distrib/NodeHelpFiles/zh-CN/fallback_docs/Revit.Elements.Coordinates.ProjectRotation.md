## 详细
“Coordinates.ProjectRotation”返回项目旋转(以度为单位)。项目旋转是用户可修改的值，可能会有很大差异。有关 Revit 中项目旋转的详细信息，请访问此 [帮助文章](https://help.autodesk.com/view/RVT/2025/CHS/?guid=GUID-C240FF71-D7D4-42C0-981C-4931C7A5E50C)。
