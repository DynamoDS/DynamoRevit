## 详细
'View3D.CreateUniqueViewName'确定是否已存在具有提供名称的视图。如果存在具有提供名称的视图，则会使用唯一名称创建新视图。否则，将返回原始视图名称。

此节点还可以与简单字符串输入一起使用，以生成用于任何图元创建的完全唯一的字符串。

在下面的示例中，从选定的三维视图生成唯一的视图名称，从字符串条目生成唯一的视图名称，并且已经证明唯一的视图名称是唯一的。
