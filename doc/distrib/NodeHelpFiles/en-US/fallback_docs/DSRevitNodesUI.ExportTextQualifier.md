## In Depth
This node schedule text qualifiers options (Quote, DoubleQuote, None).  These values will define the character that will surround text values in an exported schedule.  Example, if the Revit schedule value equeals 3, the output schedule will equal '3', if the qualifier option is set to Quote.

In this example Export Text Qualifier (Quote) is used as an input to the ScheduleView.Export node to generate a typical export option.
