## In Depth
`Units` provides a list of available unit types for use when another node requires a `DynamoUnits.Unit` input. The output includes unit types such as length, area, volume, angle, and other supported types depending on the Dynamo version.

In the example below, `Units` generates a list of Dynamo unit objects. These units can then be selected or filtered and passed into nodes that require a `DynamoUnits.Unit` input.
