## In Depth
This node is used when another node needs a DynamoUnits.Unit input. The output is a list of units, such as length, area, volume, angle, or other supported unit types, depending on the Dynamo environment/version.

In this example, the Core.Units.Units node is used to generate a list of Dynamo unit objects. These units can then be selected or filtered and passed into nodes that require a DynamoUnits.Unit input. The output of this node is a list of available units.
