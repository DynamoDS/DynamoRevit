## In Depth
`Subelement.GetParameterValue` retrieves the value of a parameter from a Revit subelement.

In the example below, all elements in the current view are selected and filtered to show only elements that contain subelements. The subelements are then used as the input to `Subelement.GetParameterValue` along with the parameter IDs. The output is the parameter value from the given subelement.
