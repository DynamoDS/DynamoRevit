## In Depth
This node lets you write DesignScript code directly inside Dynamo to create values, perform calculations, define variables, and build logic in a compact text-based format.  A Code Block can contain single values, lists, ranges, mathematical expressions, string values, function calls, and variable assignments.

In this example Code Block is used to show mathematical expression.  
For more information see link below.
https://dynamobim.org/wp-content/links/DesignScriptGuide.pdf
