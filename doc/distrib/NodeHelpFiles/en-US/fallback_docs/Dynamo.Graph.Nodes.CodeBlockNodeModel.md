## In Depth
`Code Block` provides a compact text-based format for writing DesignScript code directly in Dynamo to create values, perform calculations, define variables, and build logic. A `Code Block` can contain single values, lists, ranges, mathematical expressions, string values, function calls, and variable assignments.

In the example below, a `Code Block` demonstrates a mathematical expression. For more information, see the [DesignScript Guide](https://dynamobim.org/wp-content/links/DesignScriptGuide.pdf).
