## In Depth
Returns the coordinate system that defines the orientation of the bounding box, allowing Dynamo to represent rotated or locally oriented bounding boxes.

In this example a door is selected and its geometry is used to generate a bounding box.  This is then used as the input to BoundingBox.ContextCoordinateSystem where the coordinates system is exposed.
