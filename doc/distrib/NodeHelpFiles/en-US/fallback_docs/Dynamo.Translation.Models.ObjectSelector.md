## In Depth
This node narrows the scope of import geometry by allowing users to manually select which geometry should be passed through.

In this example CAD geometry is imported and used in the Select Geometry File Objects node. You can manually select object to be pass through like the ones shown here with ID's , 16, 27 and 33.
