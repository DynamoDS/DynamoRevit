## In Depth
This node lets you pick revision from the current Revit project (a revision picker).

In this example Select Revision node is used and "Seq. 1 - Schematic Design" is selected.  Then it's used as an input to show the revisions date and if its' been issued. 

For more information on Revisions see link.  https://help.autodesk.com/view/RVT/2025/ENU/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
