## In Depth
This node imports external geometry or model data into Dynamo so it can be viewed, filtered, or used in downstream workflows. It acts as the main entry point for bringing supported imported content into the graph, allowing Dynamo to interpret the incoming file or object and expose it for further processing.

In this example, the file North Arrows.dwg is selected and used as the input for the Import Geometry File node. The output of this node provides access to all geometry and associated data contained within the imported file.
