## In Depth
This node is used to select a quantity type (such as Length, Area, or Volume) and extract the associated unit definitions. These quantities represent the type of measurement being described, not the actual unit value. For example, Length is the quantity, while Feet or Meters are units that belong to that quantity.

In this example, the Quantities node is being used to select a value (Length, Area, Volume) and being used to extract unit definitions.
