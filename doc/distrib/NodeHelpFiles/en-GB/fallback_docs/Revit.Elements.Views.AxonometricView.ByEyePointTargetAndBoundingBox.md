## In Depth
`AxonometricView.ByEyePointTargetAndBoundingBox` creates a new axonometric 3D view in Revit using an eye point, a target point, a bounding box, a name, and an `isolateElement` Boolean value.

In the example below, points are created to define the eye point and target location for the 3D view. A bounding box is also created to control the view’s crop extents. These inputs are used with `AxonometricView.ByEyePointTargetAndBoundingBox` along with a name and Boolean value. The output is a new axonometric 3D view in Revit.
