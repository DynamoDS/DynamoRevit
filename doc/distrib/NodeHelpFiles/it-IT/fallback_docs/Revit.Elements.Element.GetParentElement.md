## In profondità
Quando si raccolgono elementi in Revit con un raccoglitore di categorie, è possibile raccogliere le famiglie nidificate che sono condivise. `Element.GetParentElement` consente di stabilire se una determinata istanza di famiglia è nidificata identificando il relativo elemento principale.

Nell'esempio seguente, tutte le istanze di famiglia "Chair-Breuer" vengono raggruppate in base all'istanza di famiglia principale.
