## In profondità
Questo nodo restituisce il tipo di connessione quando vengono specificati il nome di connessione e il percorso del file. Funziona come filtro dell'elenco di connessioni fornito nel modello di Revit associato al percorso del file.

In questo esempio, è stato fornito il percorso del file di un modello di Revit e il nome di connessione della piastra di base. In risposta, viene restituito il tipo di connessione per la piastra di base.
