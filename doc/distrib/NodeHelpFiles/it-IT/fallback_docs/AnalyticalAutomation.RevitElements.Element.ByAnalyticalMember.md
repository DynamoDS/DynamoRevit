## In profondità
Questo nodo crea un elemento strutturale fisico da un membro analitico o aggiorna un elemento fisico esistente già connesso al membro analitico. Il nodo può aggiornare la geometria o i parametri dell'elemento fisico in base al membro analitico, a seconda delle impostazioni di input. Quando il membro analitico non è associato ad alcun elemento fisico, l'elemento fisico viene creato utilizzando le informazioni del membro analitico.

In questo esempio, viene selezionato un membro analitico del progetto ACO Supermarket da un pilastro strutturale. Tale membro analitico è connesso a questo nodo e viene utilizzato un input Boolean per mostrare le impostazioni di default per l'aggiornamento della geometria e dei parametri. Il nodo produce il pilastro fisico corrispondente in base al modello analitico.
