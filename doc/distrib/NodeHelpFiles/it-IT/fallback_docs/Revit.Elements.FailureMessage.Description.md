## In profondità
Questo nodo recupera la descrizione dettagliata di un errore o di un avviso generato da una regola di Suggerimenti prestazioni. L'errore viene tradotto in un testo chiaro e leggibile, in modo che gli utenti possano comprendere meglio il problema.

In questo esempio, la regola di Suggerimenti prestazioni verifica se “View clipping is disabled”. I risultati vengono elaborati tramite il nodo FailureMessage.Description, che genera un messaggio descrittivo che spiega il motivo dell'errore.
