## In profondità
Questo nodo restituisce un elenco di nodi di connessione filtrati in base alla forma di sezione selezionata.

Quando si seleziona una serie di elementi dei dati strutturali con più nodi di connessione, in questo esempio l'elenco dei nodi di connessione viene filtrato in base a quelli che corrispondono alla forma di sezione selezionata. Gli elementi dei dati della struttura possono avere sottoelenchi di elementi, pertanto l'indice è impostato su 0, il valore di default.

Nell'esempio, vengono selezionati un gruppo di profilati a I, ala parallela, e un HSS e l'output consiste in un elenco di dizionari con le proprietà "accepted" e "rejected". Il valore accettato è il nodo di connessione filtrato in base alla forma di sezione selezionata. In questo caso, la forma di sezione è `HSS`.
