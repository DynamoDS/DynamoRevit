## In profondità
Questo nodo restituisce la designazione specifica (tipo di famiglia) dell'elemento dei dati della struttura.

In questo esempio, la selezione di un elemento dei dati di trave/pilastro strutturale fornisce il tipo di sezione. In questo caso, è una forma a W, W12x96.
