## In profondità
Questo nodo consente di selezionare l'impostazione Revision Visibility per una revisione (di seguito sono riportate le opzioni disponibili "CloudsAndTagVisible", "Tag Visible" e "Hidden"). La visibilità determina la posizione in cui comparirà la revisione selezionata (ad esempio, se viene visualizzata in nubi, etichette, tavole o è nascosta).

In questo esempio, viene creata una nuova revisione utilizzando il nodo Select Revision Visibility, valore "TagVisible".

Per ulteriori informazioni sulle revisioni, vedere il collegamento.
https://help.autodesk.com/view/RVT/2025/ITA/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
