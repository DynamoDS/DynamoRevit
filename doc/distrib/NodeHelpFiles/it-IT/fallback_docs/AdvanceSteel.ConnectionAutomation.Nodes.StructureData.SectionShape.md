## In profondità
Questo nodo restituisce la forma di sezione dell'elemento dei dati strutturali.

In questo esempio, la selezione di un elemento dei dati di trave/pilastro strutturale restituisce la forma di sezione strutturale. In questo caso, è `I-shape Parallel Flange`.
