## In profondità
Questo nodo con elenco a discesa fornisce un elenco di opzioni di Revit “View Parts Visibility” che controllano se le parti (elementi creati mediante il comando Parts, quali muri suddivisi, pavimenti e così via) sono visibili, nascoste o mostrate al posto degli elementi originali in una determinata vista di Revit.

In questo esempio, viene selezionata una vista (3D View) che è poi utilizzata come input (view) nel nodo View.SetPartsVisibility insieme a qualsiasi valore di default del nodo stesso. Gli ultimi 2 nodi mostrano l'impostazione originale di visibilità delle parti della vista e quella rivista.

Per ulteriori informazioni, vedere il collegamento seguente.
https://help.autodesk.com/view/RVT/2019/ITA/?guid=GUID-37A4B924-AB44-486F-AD27-02F414DE4683
