## In profondità
Questo nodo verifica se una determinata regola di Suggerimenti prestazioni è attualmente attivata in Revit.

In questo esempio, la regola “View clipping is disabled” è connessa al nodo PerformanceAdviserRule.Enabled. L'output viene visualizzato nel nodo Watch come true, confermando che la regola è attivata e pronta per l'utilizzo nei controlli del modello.
