## In profondità
Questo nodo recupera una vista di prospetto specifica da un contrassegno di prospetto in base ad un valore di indice specificato. Un contrassegno di prospetto può ospitare fino a quattro prospetti (uno per ogni direzione). Questo nodo consente di estrarre una delle viste di prospetto associate per un ulteriore utilizzo o modifica (0,1,2,3).

In questo esempio, vengono selezionati tutti i contrassegni di prospetto che sono poi utilizzati per l'input elevationMarker insieme a tutte le opzioni di indice 0,1,2,3. L'output mostra tutte le direzioni di prospetto possibili e un valore nullo se non esiste alcun prospetto. L'ultimo nodo visualizza il numero totale di prospetti associati al contrassegno di un determinato elemento.
