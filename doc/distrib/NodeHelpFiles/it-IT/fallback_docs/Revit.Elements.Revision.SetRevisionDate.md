Il nodo Revision.SetRevisionDate viene utilizzato per aggiornare o assegnare una data ad una revisione specifica in Revit. Consente agli utenti di automatizzare il processo di impostazione delle date di revisione.

In questo grafico, il nodo Select Revision viene utilizzato per scegliere una revisione dal progetto e viene fornito un input string (ad esempio, 08/31/2025) come nuova data. Il nodo Revision.SetRevisionDate applica quindi questo valore alla revisione selezionata.
