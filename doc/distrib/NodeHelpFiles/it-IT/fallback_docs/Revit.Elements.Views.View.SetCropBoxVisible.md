## In profondità
`View.SetCropBoxVisible` attiva/disattiva la visibilità di CropBox della vista specificata in base al valore booleano di input.

Nell'esempio seguente, le caselle CropBox sono impostate su nascoste per tutte le viste su tutte le tavole del documento di Revit corrente.
