Questo nodo recupera la curva geometrica sottostante associata ad un determinato CurveElement di Revit.

In questo esempio, viene creata una curva del modello utilizzando un punto iniziale e un punto finale, quindi inserita nel nodo CurveElement.Curve. L'output espone tale geometria della curva in modo che possa essere utilizzata per l'analisi, la modifica o come input in altri nodi di Dynamo.
