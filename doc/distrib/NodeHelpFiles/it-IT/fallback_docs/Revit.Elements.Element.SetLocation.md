## In profondità
Questo nodo sposta un elemento in una nuova posizione in base ad un nuovo punto.

In questo esempio, viene selezionato un elemento che è poi utilizzato come input nel nodo Element.SetLocation. La posizione originale degli elementi viene mostrata e utilizzata per la traslazione lungo il piano x e y, che è controllato dal dispositivo di scorrimento numerico. L'output di traslazione viene utilizzato come input geometry per Element.SetLocation. Verrà visualizzato un valore nullo, ma l'elemento verrà spostato.
