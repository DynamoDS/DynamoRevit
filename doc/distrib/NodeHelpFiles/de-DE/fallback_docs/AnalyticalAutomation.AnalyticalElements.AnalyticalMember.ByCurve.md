## Im Detail
Dieser Block erstellt ein analytisches Element aus einer Kurve.

In diesem Beispiel werden zwei Punkte definiert, indem die Positionen der Rasterschnittpunkte bei A4 und A5 im Snowdon-Tragwerksmodell extrahiert werden. Eine Ebene, L2 TOS, wird verwendet, um die vertikale Position beider Punkte festzulegen.
