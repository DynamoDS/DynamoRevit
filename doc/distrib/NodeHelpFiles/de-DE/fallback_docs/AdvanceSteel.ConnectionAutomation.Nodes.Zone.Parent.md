## Im Detail
Dieser Block ruft die übergeordnete Zone oder das übergeordnete Element einer bestimmten Zone ab.

In diesem Beispiel werden mehrere Elemente ausgewählt, die Tragwerksdaten extrahiert, der Verbindungsblock angezeigt und damit alle Zonen identifiziert, die für diesen Verbindungsblock gelten. Diese Zonen fungieren als Eingabe für den Zone.Parent-Block, in dem die Ausgabe die Tragwerksdaten jeder übergeordneten Zoneneingabe anzeigt. Die letzten Blöcke zeigen den Schnitttyp und die Position der Ergebnisse an.
