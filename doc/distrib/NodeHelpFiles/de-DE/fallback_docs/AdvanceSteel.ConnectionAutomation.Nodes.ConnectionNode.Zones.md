## Im Detail


Dieser Block gibt eine Liste der Zonen innerhalb der einzelnen Verbindungsblöcke zurück.

In diesem Beispiel wird durch Auswahl eines Trägerelements eine Liste mit Zonen zurückgegeben. Die Eingabe ist eine Liste mit Verbindungsblöcken, in der ein Verbindungsblock Tragwerksdatenelemente (wie Träger und Stützen) in einer einzigen Verbindung gruppiert.

Innerhalb eines Verbindungsblocks stellt die Zone einen bestimmten Teil des Tragwerksdatenelements dar, das an der Verbindung beteiligt ist. Die primären Zonentypen sind 'Ende' und 'Körper'.

Die Endzone stellt das Ende eines Tragwerksdatenelements dar: Hier findet die Verbindung statt.

Die Körperzone bezieht sich auf eine Position auf einem Tragwerksdatenelement, die vom Ende entfernt liegt, z. B. die Stelle, an der eine Strebe oder ein Stab, die/der in den Steg eines Trägers eingepasst wird, verbunden wird.
