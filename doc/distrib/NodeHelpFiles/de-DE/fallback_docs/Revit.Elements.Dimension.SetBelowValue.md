## Im Detail
Mit `Dimension.SetBelowValue` wird der nachstehende Wert der angegebenen Bemaßung auf die angegebene Zeichenfolge festgelegt. Bei Multi-Segmentbemaßungen wird derselbe Wert für alle Segmente festgelegt.
