## Im Detail
Mit `View.Discipline` wird die Ansichtsdisziplin der angegebenen Ansicht abgerufen. Zu den Optionen für die Ansichtsdisziplin gehören Architektur, Tragwerk, Gebäudetechnik, Elektro, Sanitär und Koordination.
