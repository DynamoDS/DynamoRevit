## Im Detail
Der Block Revision.ByName in Dynamo wird verwendet, um eine Änderung in einem Revit-Projekt zu erstellen oder darauf zuzugreifen, indem der Name angegeben wird. Dieser Block ermöglicht die Steuerung verschiedener Änderungsparameter, einschließlich Änderungsdatum, Beschreibung, Ausgabestatus, Ausgegeben von, Ausgegeben an, Sichtbarkeit und Nummerierungstyp. Durch die Nutzung dieses Blocks können Benutzer den Änderungsprozess in Revit automatisieren.

Das bereitgestellte Dynamo-Diagramm veranschaulicht die Automatisierung der Änderungserstellung, indem projektspezifische Eingaben mit dem Block Revision.ByName verknüpft werden. Wichtige Details wie Name des Änderungssatzes, Datum, Beschreibung, Berater, Kunde, Sichtbarkeitsoption und Nummerierungstyp werden als Eingaben bereitgestellt. Nach der Ausführung generiert oder aktualisiert der Block die Änderung in Revit basierend auf diesen Parametern. Dieser Arbeitsablauf reduziert die manuelle Dateneingabe auf ein Minimum, erzwingt die Standardisierung und optimiert die Änderungsverwaltung über Projektpläne hinweg.
