## Im Detail
Dieser Block gibt den Y-Achsenvektor eines Tragwerksdatenelements zurück.

In diesem Beispiel wird durch Auswahl eines Träger-/Stützenelements der Vektor (X = -1.000, Y = 0.000, Z = 0.000, Länge = 1.000) zurückgegeben.
