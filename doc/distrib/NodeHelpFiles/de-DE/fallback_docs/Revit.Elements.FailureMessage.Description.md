## Im Detail
Dieser Block ruft die detaillierte Beschreibung eines Fehlers oder einer Warnung ab, der/die durch eine Leistungsberater-Regel generiert wurde. Der Block wandelt den Fehler in klaren, lesbaren Text um, sodass Benutzer das Problem besser verstehen können.

In diesem Beispiel prüft die Leistungsberater-Regel, ob das Zuschneiden für die Ansicht deaktiviert ist. Die Ergebnisse werden durch den FailureMessage.Description-Block verarbeitet, der eine Meldung ausgibt, in der der Grund für den Fehler erläutert wird.
