## Im Detail
Dieser Dropdown-Block enthält eine Liste der Revit-Optionen für die Sichtbarkeit von Bauteilen, mit denen gesteuert wird, ob Bauteile (Elemente, die mit dem Befehl Teile erstellt wurden, z. B. unterteilte Wände, Geschossdecken usw.) sichtbar oder ausgeblendet sind oder anstelle ihrer ursprünglichen Elemente in einer bestimmten Revit-Ansicht angezeigt werden.

In diesem Beispiel wird eine Ansicht (3D-Ansicht) ausgewählt und als Eingabe (Ansicht) für den Block View.SetPartsVisibility zusammen mit den Vorgabewerten aus dem Block View Parts Visibilities verwendet. Die letzten beiden Blöcke zeigen die Sichtbarkeitseinstellung der Bauteile in der ursprünglichen und der überarbeiteten Einstellung an.
