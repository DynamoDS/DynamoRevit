## Im Detail
Dieser Block legt fest, wie Teile in einer angegebenen Revit-Ansicht angezeigt werden. Mit diesem Block können Sie die Ansicht steuern: "ShowPartsOnly", "ShowOriginalOnly", "ShowPartsAndOriginal" oder "unset".

In diesem Beispiel wird eine Ansicht (3D-Ansicht) ausgewählt und als Eingabe (Ansicht) für den Block View.SetPartsVisibility zusammen mit den Vorgabewerten aus dem Block View Parts Visibilities verwendet. Die letzten beiden Blöcke zeigen die Sichtbarkeitseinstellung der Bauteile in der ursprünglichen und der überarbeiteten Einstellung an.
