## Im Detail
Dieser Block ruft den Schweregrad eines bestimmten FailureMessage-Objekts ab. Schweregradwerte sind "Warnung", "Fehler" und "Kritisch".

In diesem Beispiel wird eine Leistungsberater-Regel ausgeführt, um zu prüfen, ob die Bedingung “Zuschneiden deaktiviert für Ansicht” erfüllt ist. Die resultierenden Fehlermeldungen werden an den FailureMessage.Severity-Block übergeben, der den Schweregrad der einzelnen Probleme angibt.
