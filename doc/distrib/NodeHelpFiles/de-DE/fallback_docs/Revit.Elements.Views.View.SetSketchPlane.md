## Im Detail
Dieser Block legt die Vorgabeskizzierebene in einer bestimmten Ansicht fest.

In diesem Beispiel werden alle Skizzierebenen aus dem Projekt abgerufen und bis zur Skizzierebene “Parking” gefiltert. Diese Skizzierebene “Parking” wird dann als Eingabe (SketchPlane) für den Block View.SetSketchPlane zusammen mit einer Ansicht (view) verwendet. Die letzten beiden Blöcke zeigen den ursprünglichen Skizzierebenennamen zusammen mit dem neuen Skizzierebenennamen an.
