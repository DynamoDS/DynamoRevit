## Im Detail
Dieser Block prüft, ob eine bestimmte Leistungsberater-Regel momentan in Revit aktiviert ist.

In diesem Beispiel ist die Regel “Zuschneiden deaktiviert für Ansicht” mit dem Block PerformanceAdviserRule.Enabled verknüpft. Die Ausgabe wird im Watch-Block als true angezeigt, was bestätigt, dass die Regel aktiviert ist und in Modellprüfungen verwendet werden kann.
