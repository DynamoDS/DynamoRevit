## Im Detail
Dieser Block gibt eine Liste von Verbindungsblöcken zurück, gefiltert nach der ausgewählten Querschnittsform.

Wenn Sie eine Verbindungsgruppe auswählen, die aus einer W-Form und einem HSS-Element besteht, wird in diesem Beispiel die Liste der Verbindungsblöcke nach den Blöcken gefiltert, die über einen Steg-Flansch-Schnitt verfügen. Tragwerksdatenelemente können untergeordnete Listen mit Elementen enthalten, sodass der Index auf 0 gesetzt wird - den Vorgabewert.

In diesem Beispiel wurden eine W-Form und ein HSS-Element ausgewählt, und die Ausgabe besteht aus einer Liste von Wörterbüchern mit den Eigenschaften "Akzeptiert" und "Abgelehnt". Der akzeptierte Wert ist der Verbindungsblock, gefiltert nach der ausgewählten Querschnittsform. In diesem Fall lautet die Querschnittsform `W24X104`.
