## Im Detail
Mit `Dimension.SetAboveValue` wird der vorstehende Wert der angegebenen Bemaßung auf die angegebene Zeichenfolge festgelegt. Bei Multi-Segmentbemaßungen wird derselbe Wert für alle Segmente festgelegt.
