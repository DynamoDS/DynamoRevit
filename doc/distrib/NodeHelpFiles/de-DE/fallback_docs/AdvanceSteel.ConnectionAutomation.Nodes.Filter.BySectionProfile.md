## Im Detail
Dieser Block gibt eine Liste von Verbindungsblöcken zurück, gefiltert nach dem ausgewählten Schnittprofil.

Wenn Sie eine Reihe von Tragwerksdatenelementen mit mehreren Verbindungsblöcken auswählen, wird die Liste der Verbindungsblöcke nach dem ausgewählten Schnittprofil gefiltert. Tragwerksdatenelemente können untergeordnete Listen mit Elementen enthalten, sodass der Index auf 0 gesetzt wird - den Vorgabewert.

In diesem Beispiel wurde eine Gruppe von W-Formen ausgewählt, und die Ausgabe besteht aus einer Liste von Wörterbüchern mit den Eigenschaften "Akzeptiert" und "Abgelehnt". Der Wert "Akzeptiert" ist der Verbindungsblock, gefiltert nach dem ausgewählten Schnittprofil. In diesem Fall lautet das Schnittprofil 'W Shapes-Column'.
