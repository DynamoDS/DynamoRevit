## Im Detail

Dieser Block gibt eine Liste von Verbindungsblöcken zurück, gefiltert nach dem ausgewählten Zonentyp: Ende oder Körper.

Wenn Sie eine Reihe von Tragwerksdatenelementen mit mehreren Verbindungsblöcken auswählen, wird in diesem Beispiel die Liste der Verbindungsblöcke mit dem ausgewählten Schnittprofil gefiltert. Tragwerksdatenelemente können untergeordnete Listen mit Elementen enthalten, sodass der Index auf 0 gesetzt wird - den Vorgabewert.

In diesem Beispiel wurde eine Gruppe von I-förmigen parallelen Flanschen ausgewählt, und die Ausgabe besteht aus einer Liste von Wörterbüchern mit den Eigenschaften "Akzeptiert" und "Abgelehnt". Der akzeptierte Wert ist der Verbindungsblock, gefiltert nach dem ausgewählten zoneType-Objekt. In diesem Fall lautet das zoneType-Objekt 'end'.
