## Im Detail
Dieser Block gibt eine Liste von Verbindungsblöcken zurück, gefiltert nach der ausgewählten Querschnittsform.

Wenn Sie eine Reihe von Tragwerksdatenelementen mit mehreren Verbindungsblöcken auswählen, wird in diesem Beispiel die Liste der Verbindungsblöcke nach den Blöcken gefiltert, die der ausgewählten Querschnittsform entsprechen. Tragwerksdatenelemente können untergeordnete Listen mit Elementen enthalten, sodass der Index auf 0 gesetzt wird - den Vorgabewert.

In diesem Beispiel wurde eine Gruppe von I-förmigen parallelen Flanschen sowie ein HSS-Element ausgewählt, und die Ausgabe besteht aus einer Liste von Wörterbüchern mit den Eigenschaften "Akzeptiert" und "Abgelehnt". Der akzeptierte Wert ist der Verbindungsblock, gefiltert nach der ausgewählten Querschnittsform. In diesem Fall lautet die Querschnittsform`HSS`.
