## Im Detail
Dieser Block erstellt ein neues Element einer geteilten Oberfläche auf einer ausgewählten Fläche eines Revit-Elements und definiert dessen Layout mit den angegebenen U- und V-Unterteilungen. Eine geteilte Oberfläche ist ein gemustertes Raster, das auf eine Fläche angewendet wird und häufig zum Positionieren von Fassadenelementen, adaptiven Bauteilen oder Paneelsystemen auf einer Formoberfläche verwendet wird.

Die U- und V-Unterteilungen bestimmen, wie viele Unterteilungen in jeder Oberflächenrichtung erfolgen, während der Rotationsparameter die Rasterausrichtung relativ zum U-V-Koordinatensystem der Oberfläche anpasst.

In diesem Beispiel wird eine Fläche ausgewählt und als Eingabe für die Oberfläche für den DividedSurface.ByFaceUVDivisions-Block zusammen mit UDivs und VDivs verwendet, die durch Schieberegler gesteuert werden. Die letzten Blöcke zeigen die Werte der geteilten Oberfläche an. Beim Ausführen dieses Beispieldiagramms müssen Sie die Revit-Warnmeldung beachten und die vorgeschlagenen Elemente löschen, damit die Raster auf der ausgewählten Oberfläche angezeigt werden.
