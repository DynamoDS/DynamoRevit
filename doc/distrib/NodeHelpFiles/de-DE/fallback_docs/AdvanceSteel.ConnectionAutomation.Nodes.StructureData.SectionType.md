## Im Detail
Dieser Block gibt die spezifische Bezeichnung (Familientyp) des Tragwerksdatenelements zurück.

In diesem Beispiel wird durch Auswahl eines Träger-/Stützendatenelements der Profiltyp bereitgestellt. In diesem Fall handelt es sich um eine W-Form, W12x96.
