## Im Detail
Dieser Block platziert Exemplare eines adaptiven Bauteils, indem die adaptiven Punkte direkt aus einer Liste von Punktkoordinaten definiert werden und der zu verwendende Familientyp angegeben wird.

In diesem Beispiel wird ein einzelner Punkt für den AdaptiveComponent.ByPoints-Block zusammen mit dem adaptiven Bauteil "Diagnostic Tripod - 1 Point.rfa" für den Eingabe-familyType bereitgestellt. Die Ausgabe ist ein adaptives Bauteilexemplar, das bei 1,1,0 der Körperfamilie platziert wird. Beachten Sie, dass "Diagnostic Tripod - 1 Point.rfa" in die Körperfamilie geladen werden muss, bevor Sie dieses Diagramm ausführen können.

Damit diese Block-Hilfe-Beispieldatei ausgeführt werden kann, müssen Sie "Diagnostics Tripod - 1 Point.rfa" in die Revit-Datei laden. Die Familie wird hier gespeichert. C:\ProgramData\Autodesk\RVT 2027\Dynamo\Samples\Data
