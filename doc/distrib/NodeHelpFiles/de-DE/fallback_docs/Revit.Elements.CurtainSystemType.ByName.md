## Im Detail
`CurtainSystemType.ByName` wählt einen Fassadensystemtyp mit dem angegebenen Namen aus dem aktuellen Revit-Dokument aus.

Im folgenden Beispiel wird der Fassadensystemtyp `5' x 10'` zurückgegeben.
