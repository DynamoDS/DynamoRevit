## Im Detail
Dieser Block verschiebt den Mittelpunkt eines bestimmten Ansichtsfensters an eine neue Position im Plan.

In diesem Beispiel wird ein neuer Plan "Dynamo Sheet" erstellt, und die Ansicht "L1 Wall Base" wird am Punkt "0,0,0" platziert.
