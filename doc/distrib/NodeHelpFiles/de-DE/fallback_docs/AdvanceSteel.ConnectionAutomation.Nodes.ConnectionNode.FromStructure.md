## Im Detail


Dieser Block gibt eine Gruppe von Tragwerksdatenelementen in einer Liste von Verbindungsblöcken zurück.

In diesem Beispiel wurde durch Auswahl eines Trägerelements der Verbindungsblock an jedem Ende des Trägers zurückgegeben.
