## Im Detail
`CeilingType.Name` gibt den Namen des angegebenen Deckentyps zurück.

Im folgenden Beispiel werden alle Deckentypen aus dem aktuellen Revit-Dokument gesammelt. Die Namen der Deckentypen werden mit `CeilingType.Name` zurückgegeben.
