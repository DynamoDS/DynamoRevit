## Im Detail
`Detaillierungsgrad` ermöglicht die Auswahl eines Ansichtsdetaillierungsgrads für die Verwendung in Grafikeinstellungen.

Die verfügbaren Detaillierungsgrade umfassen:
- Grob
- Fein
- Mittel
- Undefiniert
