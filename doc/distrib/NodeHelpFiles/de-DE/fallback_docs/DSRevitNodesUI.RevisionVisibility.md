## Im Detail
Über diesen Block können Sie die Einstellung für die Sichtbarkeit von Änderungen für eine Änderung auswählen (hier sind die verfügbaren Optionen CloudAndTagVisible, TagVisible und Hidden). Die Sichtbarkeit bestimmt, wo die ausgewählte Änderung angezeigt wird (ob sie z. B. in Wolken, Beschriftungen oder Plänen angezeigt wird oder ausgeblendet ist).

In diesem Beispiel wird eine neue Änderung mithilfe des Blocks Select Revision Visibility und dem Wert "TagVisible" erstellt.

Weitere Informationen zu Änderungen finden Sie unter dem Link.
https://help.autodesk.com/view/RVT/2025/DEU/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
