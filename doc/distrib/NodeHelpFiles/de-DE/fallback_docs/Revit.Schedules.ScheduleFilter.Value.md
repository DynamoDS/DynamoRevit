## Im Detail
`ScheduleFilter.Value` gibt den Wert zurück, der dem angegebenen Filter zugewiesen ist.

Im folgenden Beispiel wird ein Filter auf eine Türliste angewendet, der nur Elemente anzeigt, die nicht mit "OPN" enden.
