## Im Detail
Dieser Block generiert eine Revit-Referenzlinie (Referenzkurve) innerhalb eines Revit-Familiendokuments.

In diesem Beispiel definieren die von einem Zahlen-Schieberegler generierten Koordinaten die Start- und Endpunkte einer Modellkurve, die dann in diesen Block eingegeben wird. Der letzte Block in der Sequenz zeigt alle Parameter und die entsprechenden Werte für die neu erstellte Referenzlinie an. Die Funktionalität dieses Blocks ist auf Revit-Familiendokumente beschränkt.
