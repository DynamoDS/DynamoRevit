## Description approfondie
Ce nœud génère une ligne de référence Revit (courbe de référence) dans un document de la famille Revit.

Dans cet exemple, les coordonnées générées par un curseur numérique définissent les points de départ et d'arrivée d'une courbe de modèle, qui est ensuite introduite dans ce nœud. Le dernier noeud de la séquence présente tous les paramètres et leurs valeurs correspondantes pour la ligne de référence nouvellement générée. La fonctionnalité de ce nœud est limitée aux documents de la famille Revit.
