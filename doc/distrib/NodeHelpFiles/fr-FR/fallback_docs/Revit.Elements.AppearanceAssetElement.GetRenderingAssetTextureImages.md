## Description approfondie
Ce nœud récupère toutes les images de texture associées à l'actif de rendu d'un AppearanceAssetElement donné dans Revit. Un actif d'apparence définit les propriétés visuelles d'un matériau (telles que la couleur, la réflectivité, le relief ou la transparence).

Dans cet exemple, les éléments AppearanceAssetElement sont sélectionnés et utilisés en tant qu'entrée pour le nœud AppearanceAssetElement.GetRenderingAssetTextureImages.  Les résultats sont les propriétés de l'image et les chemins d'accès à l'image pour l'élément actif d'apparence donné.  Les listes vides sont affichées lorsque l'apparence actif élément n'a pas de valeur.
