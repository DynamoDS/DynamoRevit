## Description approfondie
Ce nœud vérifie si une règle de Performance Adviser est actuellement activée dans Revit.

Dans cet exemple, la règle "Le découpage de la vue est désactivé" est connectée au nœud PerformanceAdviserRule.Enabled. La sortie est indiquée comme vraie dans le nœud Watch, ce qui confirme que la règle est activée et prête à être utilisée dans les vérifications de modèle.
