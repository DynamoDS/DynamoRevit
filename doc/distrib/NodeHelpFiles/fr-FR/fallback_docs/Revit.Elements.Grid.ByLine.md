## Description approfondie
Ce noeud crée une grille basée sur une entrée de ligne.

Dans cet exemple, un point est créé et utilisé comme entrée startPoint pour le nœud Line.ByStartPointDirectLength, ainsi qu'une direction et une longueur.  La sortie est une grille.
