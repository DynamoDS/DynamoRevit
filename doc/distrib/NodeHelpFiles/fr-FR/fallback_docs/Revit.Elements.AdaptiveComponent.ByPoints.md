## Description approfondie
Ce noeud place des instances d'un composant adaptatif en définissant ses points adaptatifs directement à partir d'une liste de coordonnées de points et en spécifiant le type de famille à utiliser.

Dans cet exemple, un point unique est fourni au nœud AdaptiveComponent.ByPoints avec le composant adaptatif « Trépied de diagnostic – 1 Point.rfa » pour l’entrée familyType.  La sortie est une instance de composant adaptatif placée au niveau 1,1,0 de la famille de volumes.  Notez que le « Trépied de diagnostic – 1 Point.rfa » doit être chargé dans votre famille de volumes avant d’exécuter ce graphique.

Pour exécuter ce fichier d'exemple d'aide sur le nœud, vous devez charger "Diagnostics Tripod-1 point.rfa" dans le fichier Revit. La famille est stockée ici. C:\ProgramData\Autodesk\RVT 2027\Dynamo\Samples\Data
