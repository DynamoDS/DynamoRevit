## Description approfondie
Ce nœud récupère la description détaillée d'une défaillance ou d'un avertissement généré par une règle Performance Adviser. Il traduit la défaillance en un texte clair et lisible afin que les utilisateurs puissent mieux comprendre le problème.

Dans cet exemple, la règle Performance Adviser vérifie si l'option "Découpage de la vue est désactivée". Les résultats sont traités par le nœud FailureMessage.Description, qui génère un message descriptif expliquant la raison de l'échec.
