## Description approfondie
Ce noeud crée un élément Chemin divisé dans une famille de volumes en divisant les courbes en un nombre spécifié de segments égaux.

Dans cet exemple, 2 courbes de modèle sont créées à partir d'un point de départ et d'un point d'arrivée spécifiés, puis utilisées comme courbe en entrée pour le nœud DividedPath.ByCurvesAndDivisions. Une valeur entière définit le nombre de divisions égales le long des courbes. Le noeud final affiche les points de division obtenus le long des courbes.
