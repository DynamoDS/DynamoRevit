## Description approfondie
Ce nœud récupère une vue d'altitude spécifique à partir d'un marqueur d'altitude en fonction d'une valeur d'indice donnée.  Un marqueur d'altitude peut héberger jusqu'à quatre altitudes (une pour chaque direction). Ce noeud vous permet d'extraire l'une de ces vues d'altitude associées pour une utilisation ultérieure ou une modification (0,1,2,3).

Dans cet exemple, tous les marqueurs d'altitude sont sélectionnés et utilisés pour le marqueur d'altitude en entrée, ainsi que toutes les options d'index 0,1,2,3.  La sortie affiche toutes les directions d'altitude possibles et une valeur nulle s'il n'y a pas d'altitude.  Le dernier noeud affiche le nombre total d'altitudes associées à un marqueur d'élément donné.
