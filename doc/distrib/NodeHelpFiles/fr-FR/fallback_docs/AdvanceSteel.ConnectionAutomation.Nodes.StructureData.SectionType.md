## Description approfondie
Ce noeud renvoie la désignation spécifique (type de famille) de l'élément de données de structure.

Dans cet exemple, la sélection d'un élément de données structurelles poutre/poteau fournit le type de coupe. Dans ce cas, il s'agit d'une forme en W, 'W12x96'.
