## Description approfondie
Le nœud Revision.RevisionDate dans Dynamo permet de lire la date attribuée à une révision spécifique dans Revit. Il permet aux utilisateurs d'extraire les informations de date de révision sous forme de chaîne de texte.

Dans ce graphique, le nœud Sélectionner une révision est utilisé pour sélectionner une révision dans le projet et le nœud Revision.RevisionDate récupère sa date. Le résultat s'affiche dans le nœud Observation, ce qui permet aux utilisateurs de vérifier facilement la date de la révision choisie sans ouvrir manuellement les paramètres de révision de Revit.
