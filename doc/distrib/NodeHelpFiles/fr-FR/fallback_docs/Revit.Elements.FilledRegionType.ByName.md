## Description approfondie
'FilledRegionType.ByName' renvoie un type de zone remplie qui correspond au nom donné, si la zone en question est disponible dans le document Revit actif. Si le type de zone remplie est introuvable, une valeur nulle est renvoyée.

Dans l'exemple ci-dessous, le type de zone remplie "Gris uni" est renvoyé.
