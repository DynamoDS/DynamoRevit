## Description approfondie
Le nœud Revision.IssuedBy de Dynamo permet de lire les informations "Émis par" d'une révision dans Revit. Il aide les équipes à identifier rapidement qui a créé ou émis une révision, ce qui rend la documentation plus transparente et plus facile à suivre.

Dans ce graphique, le nœud Sélectionner une révision est utilisé pour choisir une révision dans le projet. Le nœud Revision.IssuedBy extrait la valeur "Issued By" et le nœud Watch affiche le résultat. Cette configuration permet aux utilisateurs de vérifier la propriété des révisions sans ouvrir manuellement le gestionnaire de révisions de Revit.
