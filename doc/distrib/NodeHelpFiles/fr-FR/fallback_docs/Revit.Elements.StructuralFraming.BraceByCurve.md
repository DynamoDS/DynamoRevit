## Description approfondie
Ce noeud crée un contreventement basé sur une courbe, un niveau et un type de famille d'ossature. La sortie est l'élément de contreventement nouvellement créé dans le projet.

Dans cet exemple, une ligne est tracée à partir du niveau L2 au croisement de grille A4 jusqu'au niveau L3 au croisement de grille A5. Le noeud utilise cette courbe, ainsi que le niveau et le type de famille sélectionnés, pour créer un contreventement dans le modèle structurel Snowdon.
