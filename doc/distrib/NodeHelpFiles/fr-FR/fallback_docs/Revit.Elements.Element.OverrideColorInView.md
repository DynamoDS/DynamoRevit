## Description approfondie
`Element.OverrideColorInView` remplace le motif de surface, le motif de coupe et la couleur de ligne pour l'élément donné par la couleur donnée. Ce noeud est l'équivalent des options de remplacement du bouton droit de la souris dans Revit.

Dans l'exemple ci-dessous, les occurrences de mur qui se chevauchent sont remplacées par la couleur rouge. Les occurrences de mur sont identifiées par les avertissements disponibles dans le fichier.
