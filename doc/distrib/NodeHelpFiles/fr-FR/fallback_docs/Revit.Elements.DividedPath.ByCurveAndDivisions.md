## Description approfondie
Ce noeud crée un élément Chemin divisé dans une famille de volume en divisant une courbe donnée en un nombre spécifié de segments égaux.

Dans cet exemple, une courbe modèle est créée à partir d'un point de départ et d'un point d'arrivée spécifiés, puis utilisée comme courbe en entrée pour le nœud DividedPath.ByCurveAndDivisions. Une valeur entière définit le nombre de divisions égales le long de la courbe. Le noeud final affiche les points de division obtenus le long de la courbe.
