## Description approfondie
Ce nœud crée un poteau structurel basé sur une courbe, un niveau et un type de famille de poteaux. La sortie est l'élément de colonne structurelle nouvellement créé dans le projet.

Dans cet exemple, une ligne est tracée au niveau du quadrillage croisant A4 entre les niveaux L1 35 Low et L3 dans le modèle structurel de Snowdon. Le noeud utilise cette courbe, ainsi que le niveau et le type de poteau sélectionnés, pour créer un poteau structurel.
