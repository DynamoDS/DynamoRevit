## Description approfondie


Ce noeud renvoie un groupe d'éléments de données de structure dans une liste de noeuds d'assemblage.

Dans cet exemple, la sélection d'un élément de poutre structurelle a renvoyé le noeud d'assemblage à chaque extrémité de la poutre.
