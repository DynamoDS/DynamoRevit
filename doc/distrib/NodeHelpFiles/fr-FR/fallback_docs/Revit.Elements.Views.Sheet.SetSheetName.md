## Description approfondie
`Sheet.SetSheetName` définira la feuille donnée sur un nom de feuille d'entrée (chaîne).

Dans l'exemple ci-dessous, un préfixe "(E)" est ajouté à tous les noms de feuilles dans le modèle.
