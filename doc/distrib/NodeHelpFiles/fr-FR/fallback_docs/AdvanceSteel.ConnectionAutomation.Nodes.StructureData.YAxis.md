## Description approfondie
Ce noeud renvoie le vecteur de l'axe Y d'un élément de données de structure.

Dans cet exemple, la sélection d'un élément structurel poutre/poteau renvoie Vector(X = -1.000, Y = 0.000, Z = 0.000, Length = 1.000).
