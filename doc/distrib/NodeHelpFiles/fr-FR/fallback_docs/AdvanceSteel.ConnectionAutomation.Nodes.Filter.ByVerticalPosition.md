## Description approfondie
Ce nœud filtre les éléments en entrée en fonction de leur position verticale dans le modèle. Elle vous permet de choisir d'évaluer la coordonnée Z supérieure ou inférieure (hauteur) d'un élément à des fins de comparaison ou de filtrage logique. Il est généralement utilisé dans le cadre d'un système de filtrage, souvent en conjonction avec un nœud de connexion, pour isoler les éléments au-dessus ou en dessous de certaines altitudes. Il est utile dans les flux de travail qui impliquent une analyse spatiale, tels que la séparation d'éléments de construction par niveau ou par zone.

Dans cet exemple, nous filtrons tous les éléments de données de structure sélectionnés en fonction de leur position (hauteur) dans la coordonnée z "Haut". Cela peut être utilisé pour déterminer plus en détail si la position est inférieure à un niveau, un sol ou un plafond.
