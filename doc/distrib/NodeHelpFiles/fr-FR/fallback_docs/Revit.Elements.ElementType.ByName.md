## Description approfondie
`ElementType.ByName` récupère un type d'élément à partir de Revit, en fonction du nom du type d'élément. `ElementType.ByName` est utile lorsqu'une liste déroulante n'est pas nécessaire.

Dans l'exemple ci-dessous, un type de mur correspondant au nom est renvoyé.
