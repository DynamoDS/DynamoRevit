## Description approfondie
Ce noeud renvoie le vecteur de l'axe X d'un élément de données de structure.

Dans cet exemple, la sélection d'un élément structurel poutre/poteau renvoie Vector(X = 0,000, Y = 1.000, Z = 0.000, Lengthr = 1.000).
