## Description approfondie
Le nœud Revision.SetRevisionDate permet de mettre à jour ou d'attribuer une date à une révision spécifique dans Revit. Il permet aux utilisateurs d'automatiser le processus de définition des dates de révision.

Dans ce graphique, le nœud Sélectionner une révision est utilisé pour choisir une révision dans le projet et une chaîne en entrée (par exemple, 31/08/2025) est fournie comme nouvelle date. Le nœud Revision.SetRevisionDate applique ensuite cette valeur à la révision sélectionnée.
