## Description approfondie
Ce noeud renvoie la forme de coupe de l'élément de données structurelles.

Dans cet exemple, la sélection d'un élément de données structurelles poutre/poteau renvoie la forme de coupe structurelle. Dans ce cas, il s'agit de 'I-shape Parallel Flange'.
