## Description approfondie
Ce nœud vous permet de sélectionner un type de numéro de révision parmi les options disponibles dans un projet Revit. Il est généralement utilisé pour définir la façon dont les révisions sont numérotées ou classées par ordre alphabétique dans une séquence de révision (par exemple, numériques, alphanumériques ou aucune).

Dans cet exemple, une révision est créée à l'aide du nœud Sélectionner le type de numéro de révision, valeur "Alphanumérique".

Pour plus d'informations sur les révisions, consultez le lien.
https://help.autodesk.com/view/RVT/2025/ENU/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
