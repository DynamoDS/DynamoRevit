## Description approfondie
Ce noeud récupère la zone parente ou l'élément parent d'une zone donnée.

Dans cet exemple, plusieurs éléments sont sélectionnés, les données structurelles sont extraites, le noeud de connexion est exposé et utilisé pour identifier toutes les zones qui s'appliquent à ce noeud de connexion.  Ces zones sont l'entrée du nœud Zone.Parent où la sortie affiche les données structurelles de chaque entrée de zone parente.  Les derniers noeuds indiquent le type de section et l'emplacement des résultats.
