## Description approfondie
Ce nœud vous permet de sélectionner le paramètre Visibilité de la révision pour une révision (voici les options disponibles "CloudsAndTagVisible", "Tag Visible" et "Hidden"). La visibilité détermine l'emplacement où la révision sélectionnée apparaîtra (par exemple, si elle s'affiche dans des nuages, des étiquettes, des feuilles ou si elle est masquée).

Dans cet exemple, une révision est créée à l'aide du nœud Sélectionner la visibilité de la révision, valeur "TagVisible".

Pour plus d'informations sur les révisions, consultez le lien.
https://help.autodesk.com/view/RVT/2025/ENU/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
