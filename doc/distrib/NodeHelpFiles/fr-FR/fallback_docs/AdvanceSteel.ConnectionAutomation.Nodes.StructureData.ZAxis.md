## Description approfondie
Ce noeud renvoie le vecteur de l'axe Z d'un élément de données de structure.

Dans cet exemple, la sélection d'un élément structurel poutre/poteau renvoie Vector(X = 0.000, Y = -0.000, Z = 1.000, Length = 1.000).
