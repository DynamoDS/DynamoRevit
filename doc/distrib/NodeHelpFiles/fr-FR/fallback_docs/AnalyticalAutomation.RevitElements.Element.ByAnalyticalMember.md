## Description approfondie
Ce nœud crée un élément structurel physique à partir d'un élément analytique ou met à jour un élément physique existant qui est déjà connecté au membre analytique. Le nœud peut mettre à jour la géométrie ou les paramètres de l'élément physique en fonction de l'élément analytique, en fonction des paramètres d'entrée. Lorsque l'élément analytique n'est associé à aucun élément physique, l'élément physique est créé à l'aide des informations de l'élément analytique.

Dans cet exemple, un élément analytique du projet Supermarché ACO est sélectionné à partir d'un poteau structurel. Ce membre analytique est connecté à ce nœud et une entrée booléenne est utilisée pour afficher les paramètres par défaut de mise à jour de la géométrie et des paramètres. Le nœud produit la colonne physique correspondante en fonction du modèle analytique.
