## Description approfondie
'Wall.ByFace' crée un mur sur la face de volume donnée. La ligne d'emplacement et le type du mur doivent également être indiqués.

Dans l'exemple ci-dessous, un mur de face est créé sur la surface existante sélectionnée dans le document Revit actif.
