## Podrobnosti
Uzel `View.SetDiscipline` změní disciplínu daného pohledu.

Mezi možnosti disciplíny pohledu patří:
- Architektonický
- Koordinace
- Elektroinstalace
- Mechanický
- Domovní instalace
- Konstrukční

V níže uvedeném příkladu je disciplína výchozího 3D pohledu nastavena na koordinaci.
