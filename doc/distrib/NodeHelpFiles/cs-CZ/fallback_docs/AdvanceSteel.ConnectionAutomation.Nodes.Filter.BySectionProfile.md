## Podrobnosti
Tento uzel vrací seznam uzlů přípojů filtrovaných podle vybraného profilu řezu.

Při výběru řady prvků konstrukčních dat s více uzly přípojů filtruje seznam uzlů přípojů na vybraný profil řezu. Prvky konstrukčních dat mohou obsahovat dílčí seznamy položek, proto je index nastaven na 0 – výchozí hodnotu.

V příkladu je vybrána skupina tvarů W a výstup se skládá ze seznamu slovníků s vlastnostmi „přijato“ a „odmítnuto“.  Hodnota „přijato“ je uzel přípoje filtrovaný podle vybraného profilu řezu.  V tomto případě je profil řezu 'W Shapes-Column'.
