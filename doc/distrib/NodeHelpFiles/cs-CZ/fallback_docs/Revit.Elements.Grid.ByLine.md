## Podrobnosti
Tento uzel vytvoří osnovu podle vstupu čáry.

V tomto příkladu je vytvořen bod, který se použije jako vstup počátečního bodu pro uzel Line.ByStartPointDirectLength společně se směrem a délkou.  Výstupem je mřížka.
