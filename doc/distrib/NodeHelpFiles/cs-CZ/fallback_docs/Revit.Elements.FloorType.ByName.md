## Podrobnosti
Uzel `FloorType.ByName` vrátí typ podlahy s daným názvem v aktuálním dokumentu aplikace Revit. Pokud typ podlahy neexistuje, je vrácena hodnota null.

V následujícím příkladu jsou shromážděny všechny podlahy nacházející se v aktuálním dokumentu aplikace Revit. Typy podlah se pak shromáždí získáním názvu podlahy a získáním typu podlahy pomocí uzlu `FloorType.ByName`.
