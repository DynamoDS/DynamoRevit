## Podrobnosti
Tento uzel nastaví umístění bodu ohybu ve štítku „volného konce“.

V tomto příkladu je vybrán popisek a původní body jsou extrahovány.  Hodnota X je upravena a jsou definovány nové body XYZ, které se použijí jako vstup pro umístění uzlu Tag.SetLeaderElbow společně s původně vybraným popiskem.  Výstupem je nové umístění ohybu popisků.  Viz půdorys podlaží pohledu L1.
