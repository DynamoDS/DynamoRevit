## Podrobnosti
Uzel `WallType.Width` vrací šířku daného typu stěny jako číslo.

V níže uvedeném příkladu jsou z aktuálního dokumentu aplikace Revit shromážděny všechny typy stěn spolu s šířkami (v jednotkách projektu).
