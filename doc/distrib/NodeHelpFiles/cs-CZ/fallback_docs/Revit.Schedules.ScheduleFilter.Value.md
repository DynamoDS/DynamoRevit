## Podrobnosti
Uzel `ScheduleFilter.Value` vrací hodnotu přiřazenou k danému filtru.

V níže uvedeném příkladu se použije filtr na výkaz dveří, který zobrazuje pouze prvky, pokud prvky nekončí výrazem „OPN“.
