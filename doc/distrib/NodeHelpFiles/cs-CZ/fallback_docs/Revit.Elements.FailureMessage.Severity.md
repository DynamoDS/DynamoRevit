## Podrobnosti
Tento uzel načte stupeň závažnosti daného objektu FailureMessage.  Hodnoty závažnosti jsou „Varování“, „Chyba“ a „Kritické“.

V tomto příkladu je spuštěno pravidlo nástroje Poradce pro výkon, které kontroluje podmínku „Oříznutí pohledu je zakázáno“. Výsledná hlášení o selhání jsou předána uzlu FailureMessage.Severity, který hlásí stupeň závažnosti každého problému.
