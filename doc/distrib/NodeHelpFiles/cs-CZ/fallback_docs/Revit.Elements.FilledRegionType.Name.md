## Podrobnosti
Uzel `FilledRegionType.Name` vrací název daného typu vyplněné oblasti.

V níže uvedeném příkladu jsou z aktuálního dokumentu aplikace Revit shromážděny všechny typy vyplněných oblastí a jsou vráceny názvy.
