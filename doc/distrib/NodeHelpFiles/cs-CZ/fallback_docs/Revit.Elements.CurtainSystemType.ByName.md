## Podrobnosti
Uzel `CurtainSystemType.ByName` vybere typ systému obvodového pláště s daným názvem z aktuálního dokumentu aplikace Revit.

V níže uvedeném příkladu je vrácen typ systému obvodového pláště `5' x 10'`.
