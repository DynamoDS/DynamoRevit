## Podrobnosti
`PerspectiveView.ByEyePointTargetAndBoundingBox` creates a new Revit perspective 3D view using an eye point, a target point, a bounding box, a name, and an `isolateElement` Boolean value.

In the example below, points are created to define the eye point and target location for the 3D view. A bounding box is also created to control the view’s crop extents. These inputs are used with `PerspectiveView.ByEyePointTargetAndBoundingBox` along with a name and Boolean value. The output is a new perspective 3D view in Revit.
