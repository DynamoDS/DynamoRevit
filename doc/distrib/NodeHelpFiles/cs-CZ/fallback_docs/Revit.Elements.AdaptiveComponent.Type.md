## Podrobnosti
Tento uzel načte rodinu a typ rodiny vybrané adaptivní komponenty.

V tomto příkladu jsou všechny umístěné adaptivní komponenty zadány jako vstup do uzlu AdaptiveComponent.Type. Výstupem je rodina a název typu rodiny.
