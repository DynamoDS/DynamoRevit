## Podrobnosti
Tento uzel vyhodnotí zadaný prvek a vrátí booleovskou hodnotu udávající, zda má prvek definované umístění, spolu se zprávami o výjimkách vysvětlujícími, proč by tomu tak být nemělo.

V tomto příkladu je prvek definován pomocí uzlu Element By Id a poté je předán uzlu Element.HasActualLocation. Výstup označuje hodnotu true, pokud má prvek skutečné umístění, nebo false se zprávou výjimky, pokud nemá.
