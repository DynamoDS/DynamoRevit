## Podrobnosti
Uzel `View Disciplines` umožňuje vybrat následující disciplíny, které budou pohledům přiřazeny:

- Architektura
- Koordinace
- Elektroinstalace
- Mechanické
- Domovní instalace
- Konstrukce

V níže uvedeném příkladu je disciplína výchozího 3D pohledu nastavena na koordinaci.
