## Podrobnosti
Tento uzel vytvoří náčrtovou rovinu na základě vstupu roviny.

V tomto příkladu je definována rovina a poté je připojena k uzlu SketchPlane.ByPlane, který vygeneruje odpovídající náčrtovou rovinu.
