## Podrobnosti
Uzel `Select Vertical Text Alignment` umožňuje výběr následujících možností zarovnání textových prvků:
- Dolů
- Doprostřed
- Nahoru

V níže uvedeném příkladu se vytvoří nová textová poznámka s možností zarovnání textu doprostřed.
