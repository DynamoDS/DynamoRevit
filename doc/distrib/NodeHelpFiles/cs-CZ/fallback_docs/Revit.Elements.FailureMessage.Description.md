## Podrobnosti
Tento uzel načte podrobný popis selhání nebo upozornění vygenerovaného pravidlem nástroje Poradce pro výkon. Převádí selhání do jasného a čitelného textu, aby uživatelé mohli problému lépe porozumět.

V tomto příkladu pravidlo Poradce pro výkon kontroluje, zda je oříznutí pohledu zakázáno. Výsledky jsou zpracovány prostřednictvím uzlu FailureMessage.Description, jehož výstupem je popisná zpráva vysvětlující důvod selhání.
