## Podrobnosti
Uzel `View.FilterOverrides` nabízí nastavení pro daný filtr pohledu `(ParameterFilterElement)`.

V následujícím příkladu jsou shromážděny filtry pohledů z vybraného pohledu a jsou vrácena přepsání filtrů.
