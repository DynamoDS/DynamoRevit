## Podrobnosti
Tento uzel vrací vektor osy Z prvku konstrukčních dat.

V tomto příkladu výběr prvku konstrukčního nosníku nebo sloupu vrací vektor (X = 0,000, Y = -0,000, Z = 1,000, Délka = 1,000).
