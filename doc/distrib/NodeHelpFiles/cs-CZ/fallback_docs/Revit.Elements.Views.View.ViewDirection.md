## Podrobnosti
Uzel `View.ViewDirection` vrací vektor představující směr, kterým pohled směřuje. Půdorysný pohled obvykle vede k vektoru podél osy Z.

V níže uvedeném příkladu je vrácen vektor pro aktivní pohled.
