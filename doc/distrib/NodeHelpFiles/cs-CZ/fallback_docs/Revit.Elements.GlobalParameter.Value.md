## Podrobnosti
Uzel `GlobalParameter.Value` vrací hodnotu daného globálního parametru.

V následujícím příkladu je vrácen globální parametr „Okna_Vložka severního pohledu“ a je vrácena hodnota (v jednotkách projektu).
