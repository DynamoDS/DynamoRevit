## Podrobnosti
Uzel Revision.Issued v aplikaci Dynamo se používá ke kontrole, zda je revize v aplikaci Revit označena jako vydaná. Vrací hodnotu true nebo false (booleovskou), což týmům pomáhá rychle ověřit stav revizí, aniž by bylo nutné otevírat nastavení revizí aplikace Revit.

V tomto grafu se uzel Select Revision používá k výběru revize z projektu. Uzel Revision.Issued poté zkontroluje, zda je vybraná revize vydána, a výsledek se zobrazí v uzlu Watch jako True nebo False. Díky tomu můžete snadno potvrdit stav vydání revize přímo prostřednictvím aplikace Dynamo.
