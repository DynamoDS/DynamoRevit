## Podrobnosti
Tento uzel načte základní geometrickou křivku asociovanou s daným prvkem CurveElement aplikace Revit.

V tomto příkladu je křivka modelu vytvořena pomocí počátečního a koncového bodu a poté je vložena jako vstup do uzlu CurveElement.Curve.  Výstup odhalí geometrii křivky, aby ji bylo možné použít pro analýzu, úpravy nebo jako vstup pro jiné uzly aplikace Dynamo.
