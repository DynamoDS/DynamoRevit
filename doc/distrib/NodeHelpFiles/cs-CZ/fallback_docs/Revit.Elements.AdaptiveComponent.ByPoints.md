## Podrobnosti
Tento uzel umístí instance adaptivní komponenty definováním jejích adaptivních bodů přímo ze seznamu souřadnic bodů a určením typu rodiny, který se má použít.

V tomto příkladu je předán jeden bod uzlu AdaptiveComponent.ByPoints společně sadaptivní komponentou „Diagnostic Tripod – 1 Point.rfa“ pro vstup familyType.  Výstupem je instance adaptivní komponenty umístěná v bodě 1,1,0 rodiny objemů.  Před spuštěním tohoto grafu je nutné do rodiny objemů načíst soubor „Diagnostic Tripod – 1 Point.rfa“.

Aby bylo možné spustit tento vzorový soubor nápovědy uzlu, je nutné do souboru aplikace Revit načíst soubor „Diagnostics Tripod-1 point.rfa“. Rodina je uložena zde. C:\ProgramData\Autodesk\RVT 2027\Dynamo\Samples\Data
