## Podrobnosti
`View.SetDiscipline` modifies the discipline of the given view.

View discipline options include:
- Architectural
- Coordination
- Electrical
- Mechanical
- Plumbing
- Structural

In the example below, the default 3d view's discipline is set to coordination.
