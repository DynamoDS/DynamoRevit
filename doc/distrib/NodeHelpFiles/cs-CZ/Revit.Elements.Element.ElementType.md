## Podrobnosti
Uzel `Element.ElementType` vrací typ vstupního prvku. Pokud je zadána systémová rodina, je vrácen odpovídající typ. Například při získání typu prvku pro stěnu je výsledkem typ stěny, zatímco při získání typu prvku u instance rodiny, bude získán typ rodiny.

V níže uvedeném příkladu je získán typ podlahy pro všechny vstupní instance podlahy.
