## En detalle:
`Element.UniqueId` obtiene el ID exclusivo de un elemento en forma de cadena. A diferencia de un ID de elemento, un ID exclusivo en Revit es una representación de cadena estable de un elemento en modelos compartidos y no compartidos.

En el ejemplo siguiente, se obtiene el ID exclusivo para cada elemento de muro de entrada. Los valores se combinan en listas anidadas con el elemento de muro adecuado. Esto resulta útil para exportar a Excel u otras fuentes de datos externas.
