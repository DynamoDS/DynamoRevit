## En detalle:
Este nodo devuelve el vector del eje Z de un elemento de datos de estructura.

En este ejemplo, al seleccionar un elemento de pilar o viga estructural, se devuelve el vector (X = 0.000, Y = -0.000, Z = 1.000, Longitud = 1.000).
