## En detalle:
Este nodo devuelve una lista de nodos de conexión filtrados por la forma de sección seleccionada.

Al seleccionar una serie de elementos de datos estructurales con varios nodos de conexión, este ejemplo filtra la lista de nodos de conexión en función de aquellos que coinciden con la forma de sección seleccionada. Los elementos de datos de estructura pueden tener sublistas de elementos, por lo que el índice se establece en 0, el valor por defecto.

En el ejemplo, se selecciona un grupo de alas paralelas en forma de I y un HSS y el resultado consiste en una lista de diccionarios con las propiedades "aceptado" y "rechazado". El valor "aceptado" es el nodo de conexión filtrado por la forma de sección seleccionada. En este caso, la forma de sección es un `HSS`.
