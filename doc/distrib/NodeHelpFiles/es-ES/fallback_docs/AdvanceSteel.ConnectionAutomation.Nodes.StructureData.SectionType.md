## En detalle:
Este nodo devuelve la designación específica (tipo de familia) del elemento de datos de estructura.

En este ejemplo, al seleccionar un elemento de datos de pilar o viga estructural, se obtiene el tipo de sección. En este caso, es una forma en W, W12x96.
