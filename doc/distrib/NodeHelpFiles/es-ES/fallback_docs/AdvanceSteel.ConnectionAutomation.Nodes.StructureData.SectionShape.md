## En detalle:
Este nodo devuelve la forma de sección del elemento de datos estructurales.

En este ejemplo, al seleccionar un elemento de datos de pilar o viga estructural, se obtiene la forma de sección estructural. En este caso, es`Ala paralela en I`.
