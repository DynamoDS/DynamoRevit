## En detalle:
`ElementType.Duplicate` intenta duplicar el tipo de elemento indicado con el nombre especificado. Si ya se está utilizando el nombre, se presentará un error en el nodo.

En el ejemplo siguiente, el tipo de muro `Generic - 4` se recupera del documento de Revit actual. A continuación, el tipo de muro se duplica con el nombre `Typical - 4`.
