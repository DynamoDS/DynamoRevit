## En detalle:
Este nodo genera una línea de referencia de Revit (curva de referencia) dentro de un documento de familia de Revit.

En este ejemplo, las coordenadas generadas por un control deslizante numérico definen los puntos inicial y final de una curva de modelo, que a continuación se introduce en este nodo. El último nodo de la secuencia presenta todos los parámetros y sus valores correspondientes para la línea de referencia recién generada. La funcionalidad de este nodo se limita a los documentos de familia de Revit.
