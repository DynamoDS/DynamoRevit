## En detalle:
Este nodo crea una cota midiendo la distancia entre dos caras planas de elementos de modelo seleccionadas.

En este ejemplo, la vista "L1" del plano A101 se utiliza como entrada en Dimension.ByFaces junto con superficies, una línea, un sufijo y un prefijo. La entrada de línea permite desfasar la cota con respecto a las caras seleccionadas.
