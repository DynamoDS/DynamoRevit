## En detalle:
Este nodo recupera una regla específica del asesor de rendimiento de Revit mediante su ID de regla exclusivo.

En este ejemplo, se selecciona la regla "El proyecto contiene familias y tipos que no se utilizan" y se extrae su ID de regla. Ese ID se convierte en una cadena y, a continuación, se devuelve a PerformanceAdviserRule.ById, que vuelve a identificar y recupera la misma regla para su posterior uso o automatización.
