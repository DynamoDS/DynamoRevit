## En detalle:
Este nodo recupera la descripción detallada de un error o una advertencia generados por una regla del asesor de rendimiento. Traduce el error a texto claro y legible para que los usuarios puedan comprender mejor el problema.

En este ejemplo, la regla del asesor de rendimiento comprueba si "la delimitación de vista está desactivada". Los resultados se procesan a través del nodo FailureMessage.Description, que genera un mensaje descriptivo que explica el motivo del error.
