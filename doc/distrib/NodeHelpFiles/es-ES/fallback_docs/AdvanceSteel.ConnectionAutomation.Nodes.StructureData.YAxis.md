## En detalle:
Este nodo devuelve el vector del eje Y de un elemento de datos de estructura.

En este ejemplo, al seleccionar un elemento de pilar o viga estructural, se devuelve el vector (X = -1.000, Y = 0.000, Z = 0.000, Longitud = 1.000).
