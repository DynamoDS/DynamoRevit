## En detalle:
Este nodo devuelve el vector del eje X de un elemento de datos de estructura.

En este ejemplo, al seleccionar un elemento de pilar/viga estructural, se devuelve el vector (X = 0.000, Y = 1.000, Z = 0.000, Longitud = 1.000).
