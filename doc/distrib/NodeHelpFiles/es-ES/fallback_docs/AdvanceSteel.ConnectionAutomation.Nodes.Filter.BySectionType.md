## En detalle:
Este nodo devuelve una lista de nodos de conexión filtrados por el perfil de sección seleccionado.

Al seleccionar una serie de elementos de datos estructurales con varios nodos de conexión, este ejemplo filtra la lista de nodos de conexión en función del tipo de sección seleccionado. Los elementos de datos de estructura pueden tener sublistas de elementos, por lo que el índice se establece en 0, el valor por defecto.

En el ejemplo, se selecciona un grupo de formas en W y el resultado consiste en una lista de diccionarios con las propiedades "aceptado" y "rechazado". El valor "aceptado" es el nodo de conexión filtrado por el tipo de sección seleccionado. En este caso, el tipo de sección es 'W24x104'.
