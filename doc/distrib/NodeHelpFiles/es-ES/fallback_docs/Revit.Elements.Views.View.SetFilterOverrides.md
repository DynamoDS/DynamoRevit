## En detalle:
`View.SetFilterOverrides` aplicará las modificaciones indicadas a la vista y el filtro de vista especificados.

En el ejemplo siguiente, se crea un filtro de vista para el parámetro `Calificación para incendios` en los elementos de tipo de muro. Este filtro se añade a la vista de plano de planta `L1` del modelo de muestra. Después de añadir el filtro, se agregan modificaciones a este para que los tipos de muro con una clasificación de "2 HR" pasen a ser de color rojo uniforme.
