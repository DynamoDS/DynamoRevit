## En detalle:
`CurtainPanel.HasPlane` comprueba si el panel de muro cortina de entrada es plano. Si no es plano, se devuelve un valor "False" (falso). Si es plano, se devuelve un valor "True" (verdadero).

En el ejemplo siguiente, se selecciona un elemento de muro cortina mediante el ID de elemento, junto con todos los paneles de muro cortina del elemento de muro. A continuaci√≥n, se comprueban los paneles de muro cortina para ver si son planos con `CurtainPanel.HasPlane`.
