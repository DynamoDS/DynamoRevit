## En detalle:
Este nodo crea una forma a partir de una lista de curvas y un valor booleano que define si la forma de salida es sólida o una superficie. Este nodo funciona con familias basadas en patrones, adaptativas y de masas.

En este ejemplo, se definen varias curvas de sección transversal y se introducen en el nodo Form.ByLoftCrossSections. A continuación, el nodo genera una forma 3D continua mediante la solevación a través de estas secciones.
