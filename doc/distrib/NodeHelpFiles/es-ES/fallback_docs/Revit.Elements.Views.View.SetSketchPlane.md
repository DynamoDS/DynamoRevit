## En detalle:
Este nodo define el plano de boceto por defecto en una vista determinada.

En este ejemplo, se recuperan todos los planos de boceto del proyecto y se filtran hasta el plano de boceto "Parking". Este plano de boceto "Parking" se utiliza como entrada (SketchPlane) para el nodo View.SetSketchPlane, junto con una vista (view).
