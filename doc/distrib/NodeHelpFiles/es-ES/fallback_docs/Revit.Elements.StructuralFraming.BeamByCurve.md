## En detalle:
Este nodo crea una viga estructural en el modelo de Revit a lo largo de una curva definida. La viga se define mediante la curva, el tipo de armazón seleccionado y el nivel asociado. El resultado es el elemento de viga estructural creado.

En este ejemplo, se coloca una línea en Level L2 TOS, entre las rejillas A5 y A6 del modelo estructural de Snowdon. Esta línea se proporciona como entrada de curva, y el tipo de armazón y el nivel seleccionados definen la viga. El resultado es el elemento de viga estructural recién creado.
