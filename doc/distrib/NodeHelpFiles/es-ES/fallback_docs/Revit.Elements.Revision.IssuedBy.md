## En detalle:
El nodo Revision.IssuedBy de Dynamo se utiliza para leer la información de "Emitida por" de una revisión de Revit. Ayuda a los equipos a identificar rápidamente quién ha creado o emitido una revisión, lo que hace que la documentación sea más transparente y fácil de rastrear.

En este gráfico, el nodo Select Revision se utiliza para elegir una revisión del proyecto. El nodo Revision.IssuedBy extrae el valor de "Emitida por" y el nodo Watch muestra el resultado. Esta configuración permite a los usuarios revisar la propiedad de las revisiones sin abrir manualmente el administrador de revisiones de Revit.
