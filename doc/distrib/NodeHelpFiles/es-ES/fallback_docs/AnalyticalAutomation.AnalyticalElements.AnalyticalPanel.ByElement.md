## En detalle:
Este nodo crea un panel analítico a partir de un elemento de modelo seleccionado.

En este ejemplo, se selecciona un elemento de suelo del modelo estructural Snowdon y se proporciona como entrada al nodo. El nodo puede actualizar la geometría y los parámetros del panel en función del elemento de origen, añadir o mantener una asociación con el elemento si se desea, e incluir los huecos que se encuentren en el elemento original. La salida es el panel analítico resultante.
