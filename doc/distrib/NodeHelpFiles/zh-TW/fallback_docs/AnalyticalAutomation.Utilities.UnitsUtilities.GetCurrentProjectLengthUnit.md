## 深入資訊
此節點會識別作用中 Revit 專案中用於測量長度的單位類型，並以字串值輸出其單位類型識別碼。

在此範例中，將 UnitsUtilities.GetCurrentProjectLengthUnit 節點放在圖表中，並將 Watch 節點連接到其輸出。Watch 節點會以字串 (例如英尺) 顯示目前專案的長度單位。
