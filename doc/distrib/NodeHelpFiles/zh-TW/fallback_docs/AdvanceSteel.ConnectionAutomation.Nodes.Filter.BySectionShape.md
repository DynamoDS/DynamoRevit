## 深入資訊
此節點會傳回依選取的剖面造型篩選的接頭節點清單。

選取一系列有多個接頭節點的結構資料元素時，此範例會篩選接頭節點清單，找出符合所選剖面造型的接頭節點。結構資料元素可以有項目子清單，因此索引設定為 0 (預設值)。

在此範例中，選取一組 I 形平行翼板和一個 HSS，輸出包含一個具有「accepted」和「rejected」性質的字典清單。accepted 值是依選取的剖面造型篩選的接頭節點。在此案例中，剖面造型為 `HSS`。
