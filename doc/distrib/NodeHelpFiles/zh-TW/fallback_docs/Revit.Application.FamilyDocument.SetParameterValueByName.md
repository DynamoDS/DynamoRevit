## 深入資訊
`FamilyDocument.SetParameterValueByName` 會將給定參數設定為族群文件中所選族群類型的給定值。

以下範例將目前檔案轉換為族群文件，然後將「Radius」參數的值設定為「36" Diameter」族群類型的 `Number Slider` 值。
