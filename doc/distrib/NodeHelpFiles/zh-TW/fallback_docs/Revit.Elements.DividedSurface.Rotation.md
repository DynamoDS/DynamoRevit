## 深入資訊
此節點會傳回套用到 Revit 中給定分割表面的旋轉角度 (以度為單位)。旋轉角度定義曲面上的網格樣式相對於曲面的自然 U 方向和 V 方向如何轉向。

在此範例中，選取一面並建立分割表面。使用此分割表面輸出作為 DividedSurface.Rotation 節點的輸入。輸出是套用到分割表面的旋轉係數 (以度為單位)。最後一個節點顯示分割表面的值。執行此範例圖表時，您必須觀察 Revit 警告並刪除建議的元素，讓網格出現在選取的曲面上。
