## 深入資訊
`FloorType.GetThermalProperties` 會傳回樓板類型的熱性質 (如果有)。

如需 Revit 中元素熱性質的更多資訊，請參閱此 [文章](https://help.autodesk.com/view/RVT/2024/CHT/?guid=GUID-3C378374-D360-4207-A558-3500922A452E)。
