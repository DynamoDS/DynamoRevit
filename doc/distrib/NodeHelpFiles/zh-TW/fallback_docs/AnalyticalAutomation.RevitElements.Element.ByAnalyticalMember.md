## 深入資訊
此節點會從分析構件建立實體結構元素，或更新已連接到分析構件的既有實體元素。節點可以根據分析構件更新實體元素的幾何圖形或參數 (取決於輸入設定)。如果分析構件未與任何實體元素相關聯，則使用分析構件的資訊建立實體元素。

在此範例中，從結構柱選取 ACO 超市專案中的分析構件。將該分析構件連接到此節點，使用布林輸入顯示更新幾何圖形和參數的預設設定。節點會根據分析模型產生對應的實體柱。
