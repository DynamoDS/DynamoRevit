## 深入資訊
此節點會傳回依選取的剖面造型篩選的接頭節點清單。

選取由一個 W 造型和 HSS 組成的接頭群組時，此範例會篩選接頭節點清單，找出有腹板翼板剖面的接頭節點。結構資料元素可以有項目子清單，因此索引設定為 0 (預設值)。

在此範例中，選取一個 W 造型和一個 HSS，輸出包含一個具有「accepted」和「rejected」性質的字典清單。accepted 值是依選取的剖面造型篩選的接頭節點。在此案例中，剖面造型為 `W24X104`。
