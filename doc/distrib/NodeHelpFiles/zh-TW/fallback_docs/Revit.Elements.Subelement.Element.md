## 曆긷뀯蘊뉓쮭
`Subelement.Element` returns the parent Revit element that a given subelement belongs to.

In the example below, all elements in the current view are selected and filtered to show only elements that contain subelements. The subelements are then used as the input to `Subelement.Element`. The output is a list of parent elements associated with each subelement.
