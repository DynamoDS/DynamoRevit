## 深入資訊
此節點會演算選取的元素，判定是否適合產生分析板。可選的 checkOpenings 輸入指定有效性檢查是否應包括元素內的開口。如果設定為 true，演算時會考慮開口。

在此範例中，使用 Element.ById 節點以元素 ID 定義元素，並提供給 Element.IsValidForAnalyticalPanel。輸出包括布林值 (指出元素是否有效)，以及例外訊息 (描述無法使用該元素的任何問題)。
