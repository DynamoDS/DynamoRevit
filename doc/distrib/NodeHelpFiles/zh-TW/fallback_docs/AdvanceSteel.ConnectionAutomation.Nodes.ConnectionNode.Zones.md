## 深入資訊


此節點會傳回每個接頭節點內的區域清單。

在此範例中，選取結構樑元素會傳回區域清單。輸入是接頭節點清單，其中接頭節點會將結構資料元素 (例如樑和柱) 分組在一個接頭中。

在接頭節點中，區域表示接頭中涉及的特定結構資料元素部分。主要區域類型是「end」和「body」.

End 區域表示結構資料元素的端點: 這是發生連接的位置。

Body 區域是指結構資料元素上遠離端點的位置，例如斜撐或構件框架可能連接到樑腹板的位置。
