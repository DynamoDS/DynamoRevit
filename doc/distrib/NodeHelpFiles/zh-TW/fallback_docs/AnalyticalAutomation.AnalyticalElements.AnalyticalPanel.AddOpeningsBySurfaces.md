## 深入資訊
此節點在分析板中加入開口，使用定義的曲面作為每個開口的造型。您可以視需要套用既有的輸入來更新幾何圖形、參數和關聯。

在此範例中，Snowdon 結構模型中網格交點 A4–A5 和 B4–B5 的四個點定義開口的位置，樓層 (L2 TOS) 設定其垂直位置。線條連接這些點並建立表示開口的偏移曲面。使用 L2 TOS 的樓板建立分析板，此節點會將開口曲面連結到分析板，產生對應的分析開口。
