## 深入資訊
`Material.AppearanceAssetElement` 會傳回給定材料的外觀資產 (如果有)。

外觀資產控制材料在彩現影像中的顯示方式。如需更多資訊，請參閱此 [文章](https://help.autodesk.com/view/RVT/2025/CHT/?guid=GUID-6E3C9EF0-F657-4F79-90BD-A2FB88B0467D)。
