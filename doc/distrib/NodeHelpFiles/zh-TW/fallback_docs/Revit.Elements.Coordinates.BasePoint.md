## 深入資訊
`Coordinates.BasePoint` 會以點幾何圖形傳回專案基準點。專案基準點在 Revit 中設定，可能會因專案而異。如需 Revit 中專案基準點的更多資訊，請參閱此 [說明文章](https://help.autodesk.com/view/RVT/2025/CHT/?guid=GUID-30D76259-CC67-4498-B06B-91F7517F9B65)。
