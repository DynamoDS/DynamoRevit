## 深入資訊
`Ceiling.ByOutlineTypeAndLevel` 可讓您在目前文件 (檔案) 中建立天花板。

在以下範例中，`Ceiling.ByOutlineTypeAndLevel` 節點與 `Ceiling Types` 和 `Room.FinishBoundary` 節點搭配使用，從房間的邊界建立一個簡單的天花板。
