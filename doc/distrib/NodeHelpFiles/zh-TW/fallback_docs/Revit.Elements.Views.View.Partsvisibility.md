## 深入資訊
`View.Partsvisibility` 會傳回給定視圖中零件元素的可見性設定。Revit 中的零件是一種將系統族群分割為子元件的方式。因此，有不同的方式可顯示物件。

此節點中的值包括:
- Unset - 沒有為視圖設定零件可見性。
- ShowPartsOnly - 僅顯示視圖的零件。
- ShowOriginalOnly - 顯示視圖的原始元素，而非零件。
- ShowPartsAndOriginal - 顯示視圖的原始元素和零件。
