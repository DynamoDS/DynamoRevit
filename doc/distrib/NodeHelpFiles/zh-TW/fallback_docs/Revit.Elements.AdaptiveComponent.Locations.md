## 深入資訊
此節點會擷取給定自適應元件的位置。

在此範例中，選取已放置的自適應元件及所有類型，作為 AdaptiveComponent.Locations 的 adaptiveComponent 輸入。輸出是放置的自適應元件的位置。請注意，執行此圖表之前，必須將「Diagnostic Tripod – 1 Point.rfa」載入到量體族群中。

若要執行此節點說明範例檔案，您必須將「Diagnostics Tripod-1 point.rfa」載入到 Revit 檔案中。該族群儲存在 C:\ProgramData\Autodesk\RVT 2027\Dynamo\Samples\Data
