## 深入資訊
此節點會擷取給定 Revit 連結元素的逆轉換矩陣。在 Revit 中，可透過轉換 (平移、旋轉、調整比例) 放置連結模型。此節點可讓您取得逆轉換，有效地將座標從連結模型的空間轉換回 Revit 主體模型的座標系統。

在此範例中，選取樓層 L3 的所有 Revit 連結元素，並輸入 LinkElement.LinkInverseTransform。輸出是 Revit 主體模型的座標系統。
