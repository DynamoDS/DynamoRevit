## 深入資訊

此節點會傳回依選取的區域類型 end 或 body 篩選的接頭節點清單。

選取一系列有多個接頭節點的結構資料元素時，此範例會以選取的剖面輪廓篩選接頭節點清單。結構資料元素可以有項目子清單，因此索引設定為 0 (預設值)。

在此範例中，選取一組 I 形平行翼板，輸出包含一個具有「accepted」和「rejected」性質的字典清單。accepted 值是依選取的 zoneType 篩選的接頭節點。在此案例中，zoneType 為「end」。
