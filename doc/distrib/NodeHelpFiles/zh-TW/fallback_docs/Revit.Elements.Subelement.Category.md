## 曆긷뀯蘊뉓쮭
`Subelement.Category` retrieves the category assigned to a Revit subelement. Subelements are components or pieces of a larger Revit element, such as layers, mullions, curtain panels, rebars, or nested geometry that can exist within a parent element.

In the example below, all elements in the current view are selected and filtered to show only elements that contain subelements. The subelements are then used as the input to `Subelement.Category`. The output is a list of categories associated with each subelement.
