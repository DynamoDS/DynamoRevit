## 深入資訊
此節點提供一個下拉式清單，定義在 Revit 專案中如何管理修訂編號 — 整體 (根據專案) 或個別圖紙 (根據圖紙)。

在此範例中，使用 Select Revision Numbering 節點和「PerProject」值建立新修訂。

如需修訂的更多資訊，請參閱連結。
https://help.autodesk.com/view/RVT/2025/CHT/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
