## 深入資訊
此節點會傳回依傾斜角度的最小-最大度數範圍篩選的接頭節點清單。傾斜角度是結構資料元素的 x 軸與水平軸的夾角。

在此範例中，選取一根 W 造型柱和一個斜角 HSS 斜撐，輸出包含一個具有「accepted」和「rejected」性質的字典清單。accepted 值為傾斜角度在 30 到 60 度之間的接頭節點。斜角 HSS 斜撐的傾斜角度約為 60 度，所以接受。W 造型柱的傾斜角度為 90 度，所以拒絕。
