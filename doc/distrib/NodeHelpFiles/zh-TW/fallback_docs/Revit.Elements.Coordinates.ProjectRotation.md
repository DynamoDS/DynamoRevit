## 深入資訊
`Coordinates.ProjectRotation` 會傳回專案旋轉 (以度為單位)。專案旋轉是使用者可修改的值，可能會有很大的差異。如需 Revit 中專案旋轉的更多資訊，請參閱此 [說明文章](https://help.autodesk.com/view/RVT/2025/CHT/?guid=GUID-C240FF71-D7D4-42C0-981C-4931C7A5E50C)。
