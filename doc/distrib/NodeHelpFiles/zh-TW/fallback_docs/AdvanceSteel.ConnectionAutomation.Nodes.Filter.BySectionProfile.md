## 深入資訊
此節點會傳回依選取的剖面輪廓篩選的接頭節點清單。

選取一系列有多個接頭節點的結構資料元素時，會篩選接頭節點清單，找出具有所選剖面輪廓的接頭節點。結構資料元素可以有項目子清單，因此索引設定為 0 (預設值)。

在此範例中，選取一組 W 造型，輸出包含一個具有「accepted」和「rejected」性質的字典清單。「accepted」的值是依選取的剖面輪廓篩選的接頭節點。在本案例中，剖面輪廓為「W Shapes-Column」。
