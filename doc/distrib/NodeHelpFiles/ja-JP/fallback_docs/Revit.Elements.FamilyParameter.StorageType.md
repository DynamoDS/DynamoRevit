## 詳細
`FamilyParameter.StorageType` は、指定されたパラメータが格納するデータのタイプをレポートします。

次のオプションがあります。
- 倍精度浮動小数点数
- 整数
- 文字列
- 要素 ID

次の例では、現在の Revit ドキュメントをファミリ ドキュメントに変換し、すべてのパラメータを取得します。`FamilyParameter.StorageType` は、ファミリ内のすべてのパラメータの格納タイプを取得するために使用します。
