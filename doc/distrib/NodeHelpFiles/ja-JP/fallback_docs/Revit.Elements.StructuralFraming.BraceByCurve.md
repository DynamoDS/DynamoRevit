## 詳細
このノードは、曲線、レベル、フレーム ファミリ タイプに基づいて構造ブレースを作成します。出力は、プロジェクトに新しく作成されたブレース要素です。

この例では、通芯交差 A4 のレベル L2 から通芯交差 A5 のレベル L3 まで線分が描画されます。ノードで、この曲線と、選択したレベルとファミリ タイプが使用されて、Snowdon 構造モデルにブレースが作成されます。
