## 詳細
このノードは、指定されたアダプティブ コンポーネントの位置を取得します。

この例では、配置されたアダプティブ コンポーネントとすべてのタイプが選択されて、次に、AdaptiveComponent.Locations の adaptiveComponent 入力として使用されます。出力は、配置されたアダプティブ コンポーネントの位置です。このグラフを実行する前に、「Diagnostic Tripod – 1 Point.rfa」をマス ファミリにロードする必要があります。

このノードのヘルプ サンプル ファイルを実行するには、Revit ファイルに「Diagnostics Tripod-1 point.rfa」をロードする必要があります。ファミリはここ(C:\ProgramData\Autodesk\RVT 2027\Dynamo\Samples\Data)に保存されています。
