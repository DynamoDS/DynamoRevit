## 詳細
このノードは、選択した断面形状によってフィルタされた接合ノードのリストを返します。

複数の接合ノードを持つ一連の構造データ要素を選択する場合、この例では、接合ノードのリストを選択した断面形状に一致する接合ノードでフィルタしています。構造データ要素には項目のサブリストを含めることができるため、インデックスは既定値の 0 に設定されます。

この例では、I 形平行フランジのグループと HSS が選択され、出力は「承認済み」および「却下済み」プロパティを持つディクショナリのリストで構成されます。承認される値は、選択した断面形状によってフィルタされた接合ノードです。この場合、断面形状は「HSS」です。
