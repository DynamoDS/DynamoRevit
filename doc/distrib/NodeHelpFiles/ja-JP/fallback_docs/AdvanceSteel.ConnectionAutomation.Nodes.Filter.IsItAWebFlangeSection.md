## 詳細
このノードは、選択した断面形状によってフィルタされた接合ノードのリストを返します。

W 形状と HSS で構成される接合グループを選択する場合、この例では接合ノードのリストをフィルタして、ウェブ フランジ断面を持つ接合ノードのリストにします。構造データ要素には項目のサブリストを含めることができるため、インデックスは既定値の 0 に設定されます。

この例では、W 形状と HSS が選択されており、出力は「承認済み」および「却下済み」プロパティを持つディクショナリのリストで構成されます。承認される値は、選択した断面形状によってフィルタされた接合ノードです。この場合、断面形状は「W24X104」です。
