## 詳細
`FamilyParameter.IsReporting` は、指定されたファミリ パラメータがレポート パラメータであるかどうかを示す true または false の値を返します。

次の例では、現在の Revit ドキュメント(サンプル モデルのドア ファミリ)が開いています。ファミリ内のパラメータが、レポート パラメータであるかどうかを表すブール値のリストとともに返されます。
