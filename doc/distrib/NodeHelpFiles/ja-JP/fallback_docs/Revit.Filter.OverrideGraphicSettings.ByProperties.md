## 詳細
`OverrideGraphicSettings.ByProperties` は、ビューまたはビュー テンプレート内で使用する一連の表示/グラフィックスの上書き設定を作成します。

次の例では、すべての平面図ビュー テンプレートが現在の Revit ドキュメントから収集されます。その後、ダクトの中心線を非表示にする上書き設定がビューに適用されます。
