## 詳細
このノードを使用すると、Revit プロジェクトで使用可能なオプションから改訂番号タイプを選択することができます。通常は、改訂シーケンス内での改訂の番号付け方法や文字の付け方(数字、英数字、なしなど)を定義するために使用されます。

この例では、Select Revision Number Type ノード、値「Alphanumeric」を使用して新しい改訂が作成されます。

改訂の詳細については、リンクを参照してください。
https://help.autodesk.com/view/RVT/2025/JPN/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
