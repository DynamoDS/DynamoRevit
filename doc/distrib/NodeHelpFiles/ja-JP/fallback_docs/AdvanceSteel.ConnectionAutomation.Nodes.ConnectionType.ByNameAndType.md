## 詳細
このノードは、接合名、タイプ名、ファイル パスが指定されている場合、接合タイプを返します。これは、ファイル パスに関連付けられている Revit モデルで提供された接合リストのフィルタとして機能します。

この例では、Revit モデルのファイル パス、タイプ名、ベース プレートの接合名を指定しています。応答として、ベース プレートの接合タイプが返されます。
