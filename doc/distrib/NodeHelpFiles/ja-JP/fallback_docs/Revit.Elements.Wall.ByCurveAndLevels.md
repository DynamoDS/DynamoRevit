## 詳細
指定した線分の位置および上部/下部の拘束(開始レベルと終了レベル)に指定したタイプの壁を作成します

次の例では、「Exterior - CMU on Mtl. Stud」の壁インスタンスがレベル R1 に作成され、レベル R3 で止まっています。

壁の作成の詳細については、以下を参照してください。
