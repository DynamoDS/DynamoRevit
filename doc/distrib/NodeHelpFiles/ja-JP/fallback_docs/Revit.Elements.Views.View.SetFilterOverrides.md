## 詳細
`View.SetFilterOverrides` は、指定された優先設定を、指定されたビューとビュー フィルタに適用します。

次の例では、壁タイプ要素の `Fire Rating` パラメータのビュー フィルタが作成されています。このフィルタは、サンプル モデルの「L1」平面図ビューに追加されます。フィルタを追加すると、優先設定がフィルタに追加され、「2 HR」等級の壁タイプが赤の塗り潰しに変更されます。
