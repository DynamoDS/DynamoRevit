## 詳細


このノードは、各接合ノード内のゾーンのリストを返します。

この例では、構造梁要素を選択すると、ゾーンのリストが返されます。入力されるのは接合ノードのリストです。接合ノードは、構造データ要素(梁や柱など)を 1 つの接合にグループ化します。

接合ノード内のゾーンは、接合に含まれる構造データ要素の特定の部分を表します。主なゾーン タイプは「終端」と「ボディ」です。

終端ゾーンは、構造データ要素の終端を表します。これは接合が発生する場所です。

ボディ ゾーンは、梁のウェブに組み込まれているブレースや部材フレームが接合される場所など、終端から離れた構造データ要素上の場所を指します。
