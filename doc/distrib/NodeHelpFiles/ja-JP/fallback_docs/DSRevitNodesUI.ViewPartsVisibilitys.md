## 詳細
このドロップダウン ノードは、Revit の「ビュー パーツ表示」オプションのリストを提供します。これらのオプションで、パーツ(分割された壁や床など、[パーツ]コマンドで作成された要素)を表示するか、非表示にするか、指定された Revit ビューの元の要素の代わりに表示するかをコントロールします。

この例では、ビューが選択され(3D ビュー)、View.SetPartsVisibility ノードへの入力(ビュー)として、View Parts Visibility ノードの既定値とともに使用されます。最後の 2 つのノードで、元の設定と修正された設定でのビュー パーツの表示設定が示されます。

詳細については、以下のリンクを参照してください。
https://help.autodesk.com/view/RVT/2019/JPN/?guid=GUID-37A4B924-AB44-486F-AD27-02F414DE4683
