## 詳細
このノードは、選択した断面プロファイルでフィルタされた接合ノードのリストを返します。

複数の接合ノードを持つ一連の構造データ要素を選択する場合、選択した断面プロファイルで接合ノードのリストをフィルタします。構造データ要素には項目のサブリストを含めることができるため、インデックスは既定値の 0 に設定されます。

この例では、W 形状のグループが選択され、出力は「承認済み」および「却下済み」プロパティを持つディクショナリのリストで構成されます。「承認済み」になる値は、選択した断面プロファイルによってフィルタされた接合ノードです。この場合、断面プロファイルは「W 形状-柱」です。
