## 詳細
Dynamo の Revision.IssuedBy ノードは、Revit の改訂から「発行元」情報を読み込むために使用されます。これにより、チームは誰が改訂を作成または発行したかをすばやく特定できるため、ドキュメントの透明性が高くなり、追跡が容易になります。

このグラフでは、Select Revision ノードを使用してプロジェクトから改訂を選択します。Revision.IssuedBy ノードが「発行者」の値を抽出し、Watch ノードに結果が表示されます。この設定により、ユーザは Revit の改訂マネージャを手動で開かなくても、改訂の所有権を確認することができます。
