## 詳細
このノードは、曲線から解析用部材を作成します。

この例では、Snowdon 構造モデルの A4 と A5 にある通芯交点の位置を抽出して、2 点が定義されます。レベル L2 TOS は、両方の点の垂直方向の位置を設定するために使用されます。これらの点を接続することで線分が作成され、このノードへの入力として提供されます。出力は、定義された曲線を表す解析用部材です。
