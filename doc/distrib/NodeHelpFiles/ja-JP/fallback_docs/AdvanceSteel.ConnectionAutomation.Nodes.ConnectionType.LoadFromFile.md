## 詳細
このノードは、外部の Revit (.rvt)ファイルからロードすることができる使用可能な接合タイプのリストを返します。

この例では、指定されたファイル パスから外部の Revit ファイルに関連付けられている Revit モデルで使用可能な接合のリストを返します。ロードされる接合タイプは、モデル内に既に存在してはなりません。存在する場合は、エラーが発生します。
