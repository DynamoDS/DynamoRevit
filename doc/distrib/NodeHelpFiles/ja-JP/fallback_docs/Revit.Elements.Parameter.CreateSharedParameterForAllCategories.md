## 詳細
`Parameter.CreateSharedParameterForAllCategories` は、指定された値を持つパラメータを、現在の Revit ドキュメントで使用可能なすべてのカテゴリに追加します。

次の例では、「Include in FFE」という名前の共有の Yes/No パラメータが、現在の Revit ドキュメントのすべてのカテゴリに追加されています。
