## 詳細
`DimensionType.StyleType' は、指定された寸法スタイルのスタイルを取得します。

使用可能な寸法スタイル タイプは次のとおりです。
- 長さ寸法
- 角度寸法
- 半径寸法
- 弧長寸法
- 指定点高さ
- 指定点座標
- 長さ寸法の固定
- 指定点勾配
- 直径
- 位置合わせの測点のラベル
