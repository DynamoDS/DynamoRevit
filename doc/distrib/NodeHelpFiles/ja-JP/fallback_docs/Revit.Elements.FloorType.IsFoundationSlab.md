## 詳細
`FloorType.IsFoundationSlab` は、指定された床タイプがべた基礎かどうかを示す true または false の値を返します。

次の例では、現在の Revit ドキュメントに存在するすべての床が収集されます。次に、床はタイプ別にグループ化されます。その後、`FloorType.IsFoundationSlab` によって床タイプがべた基礎かどうかがチェックされます。
