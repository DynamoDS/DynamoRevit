## 詳細
このノードは、Revit プロジェクトでの改訂番号付けの管理方法、グローバル(プロジェクトごと)またはシートごとに個別(シートごと)のいずれか、を定義するドロップダウン リストを提供します。

この例では、Select Revision Numbering ノード、値「PerProject」を使用して新しい改訂が作成されます。

改訂の詳細については、リンクを参照してください。
https://help.autodesk.com/view/RVT/2025/JPN/?guid=GUID-0F2A9A13-0A88-4D18-9DF7-C0296DE0FFE4
