## 詳細
`ReferencePoint.ByParametersOnFaceReference` は、選択したサーフェス上の指定されたパラメータでアクティブな Revit ファミリ ドキュメントの参照点要素を作成します。

注: ファミリ ドキュメントは、アダプティブ コンポーネントまたはマス ファミリである必要があります。

次の例では、ファミリ ドキュメントで選択したサーフェスの中心に参照点が作成されます。
