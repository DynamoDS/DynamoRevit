## 詳細
このノードは、パフォーマンス アドバイザ規則によって生成されたエラーまたは警告の詳細な説明を取得します。ユーザが問題をよく理解できるように、エラーを明確で読みやすいテキストに変換します。

この例では、パフォーマンス アドバイザ規則によって「ビュー クリップが無効」かどうかが確認されます。結果は FailureMessage.Description ノードで処理されます。このノードは、エラーの原因を説明する分かりやすいメッセージを出力します。
