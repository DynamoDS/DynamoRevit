## 詳細
このノードは、Revit ファミリ ドキュメント内に Revit 参照線(参照曲線)を生成します。

この例では、数値スライダで生成された座標でモデル曲線の始点と終点を定義し、次に、このノードに入力します。シーケンスの最後のノードは、新しく生成された参照線のすべてのパラメータとそれに対応する値を表します。このノードの機能は、Revit ファミリ ドキュメントに限定されます。
