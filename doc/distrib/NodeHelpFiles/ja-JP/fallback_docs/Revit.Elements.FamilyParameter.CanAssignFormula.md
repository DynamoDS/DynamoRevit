## 詳細
`FamilyParameter.CanAssignFormula` は、指定されたファミリ パラメータに式を割り当てることができるかどうかを示す true または false の値を返します。

次の例では、現在の Revit ドキュメント(サンプル モデルのドア ファミリ)が開いています。ファミリ内のパラメータが、式を割り当てられるかどうかを表すブール値のリストとともに返されます。
