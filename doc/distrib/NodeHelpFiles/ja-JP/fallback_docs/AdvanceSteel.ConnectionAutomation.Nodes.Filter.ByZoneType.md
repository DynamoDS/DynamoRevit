## 詳細

このノードは、選択したゾーン タイプ(終端またはボディ)でフィルタされた接合ノードのリストを返します。

複数の接合ノードを持つ一連の構造データ要素を選択する場合、この例では、選択した断面プロファイルで接合ノードのリストをフィルタします。構造データ要素には項目のサブリストを含めることができるため、インデックスは既定値の 0 に設定されます。

この例では、I 形平行フランジのグループが選択され、出力は「承認済み」および「却下済み」プロパティを持つディクショナリのリストで構成されます。承認される値は、選択したゾーン タイプでフィルタされた接合ノードです。この場合、zoneType は「終端」です。
