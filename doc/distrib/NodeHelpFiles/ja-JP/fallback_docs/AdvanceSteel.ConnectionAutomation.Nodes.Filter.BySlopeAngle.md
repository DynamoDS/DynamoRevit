## 詳細
このノードは、傾斜角度の最小/最大角度の範囲でフィルタされた接合ノードのリストを返します。傾斜角度は、構造データ要素の X 軸と水平軸の間の角度です。

この例では、W 形状の柱と角度付き HSS ブレースが選択されており、出力は「承認済み」および「却下済み」プロパティを持つディクショナリのリストで構成されています。使用できる値は、傾斜角度が 30 度から 60 度の範囲にある接合ノードです。角度付き HSS ブレースの勾配角度は～ 60 度であるため承認されます。W 形状柱は、勾配角度が 90 度のため拒否されます。
