## 詳細
`FamilyDocument.SetParameterValueByName` は、ファミリ ドキュメント内の選択されたファミリ タイプに対して、指定されたパラメータを指定された値に設定します。

次の例では、現在のファイルがファミリ ドキュメントに変換されます。パラメータ「Radius」の値は「36" Diameter」ファミリ タイプの `Number Slider` の値に設定されます。
