## 詳細
このノードには、使用可能な集計表フィルタ タイプ(Equal、NotEqual、GreaterThan、LessThan、Contains など)のドロップダウン リストがあります。結果では、「Exterior」で始まるすべてのドアがフィルタされます

この例では、ビューが選択され、ビューのフィールドが公開されます。「タイプ」フィールドが選択表示され、フィルタ値、集計表フィルタ タイプ「BeginsWith」とともに集計表フィルタの作成に使用されます。次に、このフィルタが、選択した集計表ビュー「WV_Wall Schedule」に適用されます。
