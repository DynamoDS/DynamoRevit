## 詳細
`LinkInstance.Document` は、指定されたリンク インスタンスのドキュメントを返します。このドキュメントを使用して、リンクから要素を収集し、他の情報についてドキュメントをクエリーできます。

次の例では、「Snowdown Towers Sample Site」リンク インスタンスがアーキテクチャ サンプル モデルに返されます。その後、リンク インスタンスのドキュメントが返されます。
