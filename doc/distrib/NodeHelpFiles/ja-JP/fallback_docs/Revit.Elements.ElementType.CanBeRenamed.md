## 詳細
`ElementType.CanBeRenamed` は、指定された要素タイプの名前を変更できるかどうかを報告します。一部のシステム ファミリは名前を変更できませんが、Revit ではユーザが作成したほとんどのファミリ定義と重複タイプを変更できます。

次の例では、対応する要素タイプを取得したモデル内のすべての壁が収集されます。次に、これらの要素タイプの名前を変更できるかどうかがチェックされます。
