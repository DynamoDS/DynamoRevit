## 詳細
パラメータ要素を指定すると、`Parameter.SetValue` は指定されたパラメータの値を入力値に更新します。

このノードの機能は `Element.SetParameterByName` と似ていますが、パラメータ要素を直接操作する点が異なります。

次の例では、(プロジェクト情報から)現在のプロジェクトの名前が返され、小文字に変換されてから、`Parameter.SetValue` によって更新されます。
