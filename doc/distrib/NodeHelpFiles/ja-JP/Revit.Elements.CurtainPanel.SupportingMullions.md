## 詳細
`CurtainPanel.SupportingMullions` は、入力されたカーテン パネルをサポートするマリオン要素を取得します。出力リスト構造は、入力パネルのマリオンを表すネストされたリストです。

次の例では、要素 ID で選択されたカーテン ウォール要素からすべてのカーテン パネルが取得され、カーテン ウォール内の各カーテン パネルをサポートするマリオンも取得されています。
