## Em profundidade
Esse nó verifica se uma determinada regra do Consultor de desempenho está ativada no momento no Revit.

Neste exemplo, a regra “O corte de vista está desativado” é conectada ao nó PerformanceAdviserRule.Enabled. A saída é mostrada no nó “Watch” como true, confirmando que a regra está ativada e pronta para uso nas verificações do modelo.
