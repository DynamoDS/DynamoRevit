## Em profundidade
Esse nó retorna uma lista de nós de conexão filtrados pela forma de corte selecionada.

Ao selecionar um grupo de conexão que consiste em uma forma W e um HSS, este exemplo filtra a lista de nós de conexão para aqueles que têm um corte de alma/mesa. Os elementos de dados estruturais podem ter sublistas de itens, portanto, o índice é definido como 0: o valor padrão.

No exemplo, uma forma em W e um corte tipo HSS são selecionados, e a saída consiste em uma lista de dicionários com propriedades “aceito” e “rejeitado”. O valor aceito é o nó de conexão filtrado pela forma de corte selecionada. Neste caso, a forma de corte é “W24X104”.
