## Em profundidade
Esse nó recupera a descrição detalhada de uma falha ou aviso gerado por uma regra do Consultor de desempenho. Ele converte a falha em texto claro e legível para que os usuários possam entender melhor o problema.

Neste exemplo, a regra do Consultor de desempenho verifica se“O corte de vista está desativado”. Os resultados são processados por meio do nó FailureMessage.Description, que gera uma mensagem descritiva explicando o motivo da falha.
