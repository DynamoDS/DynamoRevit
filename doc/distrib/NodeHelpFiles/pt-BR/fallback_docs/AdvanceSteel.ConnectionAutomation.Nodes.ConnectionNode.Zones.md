## Em profundidade


Esse nó retorna uma lista de zonas em cada nó de conexão.

Neste exemplo, a seleção de um elemento de viga estrutural retorna uma lista de zonas. A entrada é uma lista de nós de conexão, na qual um nó de conexão agrupa os elementos de dados estruturais (como vigas e colunas) em uma única conexão.

Em um nó de conexão, a zona representa uma parte específica do elemento de dados estruturais envolvido na conexão. Os tipos de zonas principais são “extremidade” e “corpo”.

A zona final representa a extremidade de um elemento de dados estruturais: é aqui que a conexão ocorre.

A zona do corpo refere-se a uma localização em um elemento de dados estruturais afastada da extremidade, como, por exemplo, onde um contraventamento ou uma estrutura do membro se pode conectar à alma de uma viga.
