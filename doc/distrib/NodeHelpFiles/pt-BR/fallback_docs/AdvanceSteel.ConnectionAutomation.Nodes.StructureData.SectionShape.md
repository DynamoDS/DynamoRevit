## Em profundidade
Esse nó retorna a forma de corte do elemento de dados estruturais.

Neste exemplo, a seleção de um elemento de dados de viga/coluna estrutural retorna a forma do corte estrutural. Neste caso, é “Mesa paralela com forma em I”.
