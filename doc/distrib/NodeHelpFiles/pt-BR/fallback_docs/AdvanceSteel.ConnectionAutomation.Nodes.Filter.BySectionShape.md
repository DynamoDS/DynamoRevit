## Em profundidade
Esse nó retorna uma lista de nós de conexão filtrados pela forma de corte selecionada.

Ao selecionar uma série de elementos de dados estruturais com vários nós de conexão, este exemplo filtra a lista de nós de conexão para aqueles que correspondem à forma de corte selecionada. Os elementos de dados estruturais podem ter sublistas de itens, portanto, o índice é definido como 0: o valor padrão.

No exemplo, um grupo de mesas paralelas em forma de I e um HSS são selecionados, e a saída consiste em uma lista de dicionários com propriedades "aceito" e "rejeitado". O valor aceito é o nó de conexão filtrado pela forma de corte selecionada. Neste caso, a forma de corte é um “HSS”.
