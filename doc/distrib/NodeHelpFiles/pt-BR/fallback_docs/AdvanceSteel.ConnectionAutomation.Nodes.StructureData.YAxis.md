## Em profundidade
Esse nó retorna o vetor do eixo Y de um elemento de dados estruturais.

Neste exemplo, a seleção de um elemento estrutural de viga/coluna retorna o vetor (X = -1,000, Y = 0,000, Z = 0,000, Comprimento = 1,000).
