## Em profundidade
O nó Revision.IssuedBy no Dynamo é usado para ler as informações “Emitido por” de uma revisão no Revit. Ele ajuda as equipes a identificar rapidamente quem criou ou emitiu uma revisão, tornando a documentação mais transparente e fácil de rastrear.

Neste gráfico, o nó Selecionar revisão é usado para escolher uma revisão do projeto. O nó Revision.IssuedBy extrai o valor “Emitido por” e o nó Watch exibe o resultado. Essa configuração permite que os usuários revisem a propriedade da revisão sem abrir o gerenciador de revisões do Revit manualmente.
