## Em profundidade
Esse nó cria um elemento estrutural físico com base em um membro analítico ou atualiza um elemento físico existente que já está conectado ao membro analítico. O nó pode atualizar a geometria ou os parâmetros do elemento físico com base no membro analítico, dependendo das configurações de entrada. Quando o membro analítico não está associado a nenhum elemento físico, o elemento físico é criado usando as informações do membro analítico.

Neste exemplo, um membro analítico do projeto ACO Supermarket é selecionado em uma coluna estrutural. Esse membro analítico está conectado a esse nó e uma entrada booleana é usada para mostrar as configurações padrão para atualizar a geometria e os parâmetros. O nó produz a coluna física correspondente com base no modelo analítico.
