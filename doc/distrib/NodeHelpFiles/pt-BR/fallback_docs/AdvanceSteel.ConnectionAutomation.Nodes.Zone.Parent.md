## Em profundidade
Esse nó recupera a zona ou o elemento principal de uma determinada zona.

Neste exemplo, são selecionados vários elementos, são extraídos os dados estruturais e o nó de conexão é exposto e usado para identificar todas as zonas que se aplicam a esse nó de conexão. Essas zonas são a entrada para o nó Zone.Parent em que a saída exibe os dados estruturais de cada entrada da zona principal. Os últimos nós mostram o tipo de seção e a localização dos resultados.
