## Em profundidade
Esse nó permite que você seja muito específico sobre com qual parte de um elemento do Revit você deseja trabalhar. Ao contrário dos nós que selecionam um elemento inteiro (como uma parede ou um piso), esse nó permite selecionar uma face ou aresta específica daquele elemento.

Neste exemplo, selecionamos um elemento aninhado dentro de uma parede cortina.
