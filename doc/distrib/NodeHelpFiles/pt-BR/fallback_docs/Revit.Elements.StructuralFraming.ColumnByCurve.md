## Em profundidade
Esse nó cria uma coluna estrutural com base em uma curva, um nível e um tipo de família de colunas. A saída é o elemento de coluna estrutural recém-criado no projeto.

Neste exemplo, é desenhada uma linha no cruzamento do eixo A4 entre os níveis L1 35 baixo e L3 no modelo estrutural Snowdon. O nó usa essa curva, juntamente com o nível e o tipo de coluna selecionados, para criar uma coluna estrutural.
