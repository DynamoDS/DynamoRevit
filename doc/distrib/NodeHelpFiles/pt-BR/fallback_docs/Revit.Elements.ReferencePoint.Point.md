## Em profundidade
`ReferencePoint.Point` retorna a geometria do ponto de referência especificado.

No exemplo abaixo, é criado um ponto de referência no centro de uma superfície selecionada no documento da família. A geometria do ponto de referência é retornada com `ReferencePoint.Point`. Além disso, os dois tipos de objeto são fornecidos para revisão.
