## Em profundidade
Esse nó retorna o vetor do eixo Z de um elemento de dados estruturais.

Neste exemplo, a seleção de um elemento estrutural de viga/coluna retorna o vetor (X = 0,000, Y = -0,000, Z = 1,000, Comprimento = 1,000).
