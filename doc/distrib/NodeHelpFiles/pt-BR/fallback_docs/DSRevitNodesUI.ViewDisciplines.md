## Em profundidade
`View Disciplines` permite a seleção das seguintes disciplinas para atribuir às vistas:

- Arquitetura
- Coordenação
- Elétrica
- Mecânica
- Hidráulica
- Estrutural

No exemplo abaixo, a disciplina da vista 3D padrão está definida como coordenação.
