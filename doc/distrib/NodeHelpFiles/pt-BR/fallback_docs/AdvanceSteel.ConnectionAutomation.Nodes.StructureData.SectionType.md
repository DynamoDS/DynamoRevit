## Em profundidade
Esse nó retorna a designação específica (tipo de família) do elemento de dados estruturais.

Neste exemplo, a seleção de um elemento de dados estruturais de viga/coluna fornece o tipo de corte. Neste caso, é uma forma em W, W12x96.
