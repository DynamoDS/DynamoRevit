## Em profundidade
Esse nó retorna o nome do tipo de conexão associado às conexões vinculadas ao elemento de dados de estrutura.

Neste exemplo, foi selecionada uma coluna com três conexões: uma placa base, uma conexão de momento e uma conexão parafusada.
