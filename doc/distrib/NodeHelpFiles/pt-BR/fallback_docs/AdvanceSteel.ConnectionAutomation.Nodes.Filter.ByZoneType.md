## Em profundidade

Esse nó retorna uma lista de nós de conexão filtrados pelo tipo de zona selecionado: extremidade ou corpo.

Ao selecionar uma série de elementos de dados estruturais com vários nós de conexão, este exemplo filtra a lista de nós de conexão com o perfil de corte selecionado. Os elementos de dados estruturais podem ter sublistas de itens, portanto, o índice é definido como 0: o valor padrão.

No exemplo, um grupo de mesas paralelas em forma de I é selecionado, e a saída consiste em uma lista de dicionários com propriedades “aceito” e “rejeitado”. O valor aceito é o nó de conexão filtrado pelo zoneType selecionado. Neste caso, o zoneType é “extremidade”.
