## Em profundidade
Esse nó retorna uma lista de nós de conexão filtrados pelo perfil de corte selecionado.

Ao selecionar uma série de elementos de dados estruturais com vários nós de conexão, filtra a lista de nós de conexão para o perfil de corte selecionado. Os elementos de dados estruturais podem ter sublistas de itens, portanto, o índice é definido como 0: o valor padrão.

No exemplo, um grupo de formas em W é selecionado e a saída consiste em uma lista de dicionários com as propriedades “aceito” e “rejeitado”. O valor de “aceito” é o nó de conexão filtrado pelo perfil de corte selecionado. Neste caso, o perfil de corte é “Formas em W – Coluna”.
