## Em profundidade
Esse nó recupera todas as imagens de textura associadas ao recurso de renderização de um determinado AppearanceAssetElement no Revit. Um recurso de aparência define as propriedades visuais de um material (como cor, reflexibilidade, saliência ou transparência).

Neste exemplo, os AppearanceAssetElement são selecionados e usados como entrada para o nó AppearanceAssetElement.GetRenderingAssetTextureImages. Os resultados são as propriedades da imagem e os caminhos da imagem para o elemento de recurso de aparência especificado. As listas vazias são exibidas quando o elemento do recurso de aparência não tem valor.
