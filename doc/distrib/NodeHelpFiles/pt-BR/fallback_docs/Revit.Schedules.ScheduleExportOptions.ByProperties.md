## Em profundidade
Esse nó controla como uma tabela do Revit é exportada.

Neste exemplo, columnHeaders (OneRow), fieldDelimiter (em branco), headersFootersBlanks (booleano), textQualifier (citação), título (booleano) são usados como entradas para criar uma opção de exportação de tabela consistente. É possível usar essa saída pelo nó ScheduleView.Export.
