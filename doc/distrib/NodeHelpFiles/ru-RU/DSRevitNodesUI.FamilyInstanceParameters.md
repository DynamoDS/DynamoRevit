## Подробности
Узел `Get Family Parameter` выбирает параметр в виде строки из выбранного экземпляра семейства или типоразмера семейства.

Полученную строку можно использовать с узлами `Element.GetParameterValueByName` и `Element.SetParameterByName`.

 В приведенном ниже примере из заданного семейства дверей возвращается параметр «Марка типоразмера».
