## Подробности
Узел Revision.SetIssuedBy в Dynamo используется для задания или обновления значения «Кем выдано» для изменения в Revit. Это позволяет автоматизировать контроль изменений, регистрируя выдавшего изменения пользователя, что обеспечивает понятность и согласованность документации без редактирования сведений в Revit вручную.

На этом графике узел Select Revision используется для выбора требуемого изменения, а входной строковое значение (например, ABC) предоставляет имя издателя. Затем узел Revision.SetIssuedBy применяет это значение к выбранному изменению, обновляя поле «Кем выдано» непосредственно в модели Revit.
