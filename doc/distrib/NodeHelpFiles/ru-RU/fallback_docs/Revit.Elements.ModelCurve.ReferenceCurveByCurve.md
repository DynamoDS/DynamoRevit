## Подробности
Этот узел создает опорную линию Revit (опорную кривую) в документе семейства Revit.

В этом примере координаты, сгенерированные с помощью числового регулятора, определяют начальную и конечную точки кривой модели, которая затем передается в этот узел. Последний узел в последовательности представляет все параметры и соответствующие им значения для вновь созданной опорной прямой. Функции этого узла ограничены документами семейства Revit.
