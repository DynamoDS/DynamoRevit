## Подробности
Узел Revision.SetIssuedTo используется для установки или обновления значения «Кому выдано» изменения в Revit. Это позволяет автоматизировать процесс регистрации получателя изменений, обеспечивая точность и согласованность проектной документации без ввода данных в Revit вручную.

На этом графике узел Select Revision используется для выбора требуемого изменения, а входные строковые данные (например, XYZ) определяют получателя. Затем узел Revision.SetIssuedTo применяет это значение к выбранному изменению, обновляя поле «Кому выдано» непосредственно в модели Revit.
