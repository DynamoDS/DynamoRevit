## Подробности
Этот узел возвращает список узлов соединений, отфильтрованных по минимальному/максимальному диапазону градусов угла наклона. Угол наклона — это угол между осью X элемента данных конструкции и вертикальной осью.

В примере выбраны W-образная колонна и угловой раскос HSS, а также выводится список словарей со свойствами «Принято» и «Отклонено». Значение «Принято» представляет собой узел соединения с углом наклона в диапазоне от 30 до 60 градусов. Угловой раскос HSS имеет угол наклона ~30 градусов, поэтому принимается. W-образная колонна имеет угол наклона 0 градусов и отклоняется.
