## Подробности

Этот узел возвращает список узлов соединений, отфильтрованных по выбранному типу зоны: конец или тело.

При выборе ряда элементов данных конструкций с несколькими узлами соединений в этом примере список узлов соединений фильтруется по выбранному профилю сечения. Элементы данных конструкции могут иметь подсписки элементов, поэтому устанавливается индекс 0 — значение по умолчанию.

В примере выбрана группа двутавровых параллельных полок, а также выводится список словарей со свойствами «Принято» и «Отклонено». Значение «Принято» представляет собой узел соединения, отфильтрованный по выбранному типу зоны. В данном случае типом зоны является значение «Конец».
