## Подробности
Этот узел возвращает число делений V, примененных к заданной разделенной поверхности в Revit. Сетка определяется в двух направлениях (U и V), а деления V определяют, на сколько сегментов делится поверхность вдоль одной оси (обычно соответствующей параметрическому направлению V поверхности). В среде концептуальных форм Revit или семействах адаптивных компонентов разделенная поверхность представляет собой сетку с образцом, нанесенную на грань (например, стену, крышу или грань формы).

В этом примере грань выбрана и используется для создания разделенной поверхности. Эта разделенная поверхность является входным параметром для узла DividedSurface.VDivisions. В последних узлах отображаются другие значения разделенной поверхности. При запуске этого примера графика необходимо просмотреть предупреждение Revit и удалить предлагаемые элементы, чтобы сетки появились на выбранной поверхности. Количество делений V, применяемых к разделенной поверхности.
