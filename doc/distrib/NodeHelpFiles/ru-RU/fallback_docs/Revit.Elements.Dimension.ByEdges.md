## Подробности
Этот узел создает размер на основе одного или нескольких опорных ребер на заданном виде.

В этом примере вид L1 с листа A101 используется в качестве входных данных для узла Dimension.ByEdges вместе со свойством referenceCurves, линией, суффиксом и префиксом. Ввод линии позволяет смещать размер от выбранных граней. Последние два узла содержат значения размеров и их префиксы.
