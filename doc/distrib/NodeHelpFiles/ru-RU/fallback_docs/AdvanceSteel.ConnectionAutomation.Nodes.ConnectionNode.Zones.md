## Подробности


Этот узел возвращает список зон в каждом узле соединения.

В этом примере при выборе элемента несущей балки возвращается список зон. Входные данные представляют собой список узлов соединений, в котором узел соединения группирует элементы данных конструкции (например, балки и колонны) в одно соединение.

В узле соединения зона представляет собой определенную часть элемента данных конструкции, участвующего в соединении. Основные типы зон — «Конец» и «Тело».

Зона «Конец» представляет собой конец элемента данных конструкции: место, где находится соединение.

Зона «Тело» — это часть элемента данных конструкции, которая не является концом, например, место соединения раскоса или элемента каркаса со стенкой балки.
