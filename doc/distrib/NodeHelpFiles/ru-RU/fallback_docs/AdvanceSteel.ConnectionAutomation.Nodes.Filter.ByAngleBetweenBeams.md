## Подробности
Этот узел возвращает список узлов соединений, отфильтрованных по минимальному/максимальному диапазону градусов.

В примере выбраны W-образная колонна и угловой раскос HSS, а также выводится список словарей со свойствами «Принято» и «Отклонено». Значение «Принято» представляет собой узел соединения, находящийся в диапазоне от 0 до 60 градусов (этот узел измеряет от оси X). Если максимальное значение равно 45 градусам, узел соединения отклоняется. Для параметров indexFirst и indexSecond устанавливается значение «Использовать уровни», чтобы привести их в соответствие со структурой входных данных.
