## Подробности
Этот узел задает роль в конструкции для аналитической панели. Узел обновляет выбранную панель в соответствии с выбранной ролью без изменения ее геометрии или других параметров.

В этом примере аналитической панели, созданной из элемента физической модели в модели конструкции Snowdon, назначена роль перекрытия в конструкции.
