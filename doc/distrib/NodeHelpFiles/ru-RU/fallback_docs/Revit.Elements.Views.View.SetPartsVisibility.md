## Подробности
Этот узел задает способ отображения деталей на указанном виде Revit. Этот узел позволяет управлять отображением вида; ShowPartsOnly, ShowOriginalOnly, ShowPartsAndOriginal или unset.

В этом примере выбран вид (3D View), он используется в качестве входных данных (view) для узла View.SetPartsVisibility вместе со значением по умолчанию из узла View Parts Visibilities. В последних двух узлах отображаются исходные и измененные параметры видимости видовых деталей.
