## Подробности
Этот узел возвращает список узлов соединений, отфильтрованных по выбранной форме сечения.

При выборе группы соединений, состоящих из W-образной формы и HSS, в данном примере список узлов соединений фильтруется по наличию сечения стенки и полки. Элементы данных конструкции могут иметь подсписки элементов, поэтому устанавливается индекс 0 — значение по умолчанию.

В примере выбраны W-образная форма и HSS, а также выводится список словарей со свойствами «Принято» и «Отклонено». Значение «Принято» представляет собой узел соединения, отфильтрованный по выбранной форме сечения. В данном случае формой сечения является значение «W24X104».
