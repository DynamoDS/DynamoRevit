## Подробности
Этот узел извлекает родительскую зону или родительский элемент заданной зоны.

В этом примере выбрано несколько элементов, извлечены структурные данные, показан узел соединения: эти данные используются для идентификации всех зон, относящихся к этому узлу соединения. Эти зоны являются входными данными для узла Zone.Parent, в котором на выходе отображаются структурные данные каждой входной родительской зоны. В последних узлах отображается тип сечения и расположение результатов.
