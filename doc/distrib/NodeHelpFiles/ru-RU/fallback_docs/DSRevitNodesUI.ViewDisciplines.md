## Подробности
Узел `View Disciplines` позволяет выбрать категории из следующего списка для назначения видам:

- архитектурный;
- координация;
- электрический;
- механический;
- сантехника;
- структурный.

В приведенном ниже примере для категории 3D-вида по умолчанию задано значение «Координация».
