## Подробности
Узел Revision.IssuedTo в Dynamo используется для считывания сведений о том, кому выдана информация из изменения в Revit. Это помогает группам отслеживать, кому было выдано изменение.

На этом графике узел Select Revision используется для выбора определенного изменения в проекте. Затем узел Revision.IssuedTo извлекает поле «Выдано», и результат отображается в узле Watch.
