## Подробности
Этот узел возвращает список узлов соединений, отфильтрованных по минимальному/максимальному диапазону градусов угла наклона. Угол наклона — это угол между осью X элемента данных конструкции и горизонтальной осью.

В примере выбраны W-образная колонна и угловой раскос HSS, а также выводится список словарей со свойствами «Принято» и «Отклонено». Значение «Принято» представляет собой узел соединения с углом наклона в диапазоне от 30 до 60 градусов. Угловой раскос HSS имеет угол уклона ~60 градусов, поэтому принимается. W-образная колонна имеет угол уклона 90 градусов и отклоняется.
