## Подробности
Этот узел восстанавливает расположение спецификации на листе.

В этом примере спецификация размещается на листе, а затем вводится в узел SheduleOnSheet.SetLocation вместе с новой точкой. Затем спецификация перемещается на том же листе. Последний узел получает новое положение спецификации.
