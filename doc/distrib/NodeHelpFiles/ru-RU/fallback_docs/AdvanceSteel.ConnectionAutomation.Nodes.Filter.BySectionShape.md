## Подробности
Этот узел возвращает список узлов соединений, отфильтрованных по выбранной форме сечения.

При выборе ряда элементов данных конструкции с несколькими узлами соединения в этом примере список узлов соединений фильтруется по выбранной форме сечения. Элементы данных конструкции могут иметь подсписки элементов, поэтому устанавливается индекс 0 — значение по умолчанию.

В примере выбрана группа двутавровых параллельных полок и HSS, а также выводится список словарей со свойствами «Принято» и «Отклонено». Значение «Принято» представляет собой узел соединения, отфильтрованный по выбранной форме сечения. В данном случае формой сечения является значение «HSS».
