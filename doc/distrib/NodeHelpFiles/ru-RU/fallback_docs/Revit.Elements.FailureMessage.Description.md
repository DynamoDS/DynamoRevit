## Подробности
Этот узел извлекает подробное описание сбоя или предупреждения, созданного правилом Консультанта по производительности. Он преобразует ошибку в понятный и удобный для чтения текст, чтобы пользователи могли лучше понять суть проблемы.

В этом примере правило Консультанта по производительности проверяет, отключена ли подрезка вида. Результаты обрабатываются с помощью узла FailureMessage.Description, который выводит описательное сообщение с объяснением причины сбоя.
