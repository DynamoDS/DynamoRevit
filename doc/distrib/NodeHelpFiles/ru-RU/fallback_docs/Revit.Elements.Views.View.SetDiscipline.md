## Подробности
Узел `View.SetDiscipline` изменяет категорию заданного вида.

Варианты категорий вида:
- архитектурный;
- координация;
- электрический;
- механический;
- сантехника;
- структурный.

В приведенном ниже примере для категории 3D-вида по умолчанию задано значение «Координация».
