## Подробности
Этот узел создает элемент разделенной траектории в семействе формообразующих элементов путем разделения кривых на заданное количество равных сегментов.

В этом примере на основе заданных начальной и конечной точек создаются две кривые модели, которые затем используются в качестве входной кривой для узла DividedPath.ByCurvesAndDivisions. Целое значение определяет число равных делений вдоль кривых. Конечный узел показывает результирующие точки разделения вдоль кривых.
