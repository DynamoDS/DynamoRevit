## Подробности
Этот узел возвращает список узлов соединений, отфильтрованных по выбранному профилю сечения.

При выборе ряда элементов данных конструкции с несколькими узлами соединений в этом примере список узлов соединений фильтруется по выбранному типу сечения. Элементы данных конструкции могут иметь подсписки элементов, поэтому устанавливается индекс 0 — значение по умолчанию.

В примере выбрана группа W-образных форм, а также выводится список словарей со свойствами «Принято» и «Отклонено». Значение «Принято» представляет собой узел соединения, отфильтрованный по выбранному типу сечения. В данном случае типом сечения является значение «W24x104».
