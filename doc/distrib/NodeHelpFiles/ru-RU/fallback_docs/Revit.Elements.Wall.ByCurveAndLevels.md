## Подробности
Создание стены заданного типа в указанном линейном местоположении и с заданной зависимостью сверху/снизу (начальный и конечный уровни)

В приведенном ниже примере экземпляр стены «Наружная часть — бетонный блок на металлической обрешетке» создается на уровне R1 и заканчивается на уровне R3.

Дополнительные сведения о создании стен см. по ссылке ниже.
https://help.autodesk.com/view/RVT/2025/RUS/?guid=GUID-30D3A26F-42FF-40FA-83D2-B43EA67F3ECD
