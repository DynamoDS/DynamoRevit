## Подробности
Этот узел оценивает заданную поверхность и определяет ее допустимость для использования при определении аналитической панели. Допустимая поверхность обычно является плоской, непрерывной и подходит для преобразования в аналитическое представление в среде аналитической модели Revit.

В этом примере собираются грани элемента перекрытия из проекта, а верхняя грань предоставляется узлу в качестве входных данных. Узел возвращает логический результат, указывающий, соответствует ли выбранная поверхность требованиям для создания аналитической панели, а также дополнительное сообщение с описанием всех проблем, обнаруженных в ходе проверки.
