## Подробности
Этот узел возвращает список доступных типов соединений, которые можно загрузить из внешнего файла Revit (RVT).

В этом примере возвращается список соединений, доступных в модели Revit, которая связана с внешним файлом Revit, по указанному пути к файлу. Обратите внимание, что загруженные типы соединений не должны существовать в модели, иначе выдается ошибка.
