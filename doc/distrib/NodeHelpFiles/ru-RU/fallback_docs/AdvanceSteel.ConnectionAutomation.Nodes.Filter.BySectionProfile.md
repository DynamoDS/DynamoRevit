## Подробности
Этот узел возвращает список узлов соединений, отфильтрованных по выбранному профилю сечения.

При выборе ряда элементов данных конструкции с несколькими узлами соединений список узлов соединений фильтруется по выбранному профилю сечения. Элементы данных конструкции могут иметь подсписки элементов, поэтому устанавливается индекс 0 — значение по умолчанию.

В примере выбрана группа W-образных форм, а также выводится список словарей со свойствами «Принято» и «Отклонено». Принимается узел соединения, отфильтрованный по выбранному профилю сечения. В данном случае профилем сечения является значение «W Формы-Колонна».
