## Подробности
Этот узел возвращает список узлов соединений, отфильтрованных по минимальному/максимальному диапазону градусов и выбранной оси.

В примере выбраны колонна W-образная колонна и угловой раскос HSS, а также выводится список словарей со свойствами «Принято» и «Отклонено». Значение «Принято» представляет собой узел соединения, находящийся в диапазоне от 0 до 60 градусов при сравнении оси X обоих элементов. Если максимальное значение равно 45 градусам, узел соединения отклоняется. Для параметров indexFirst и indexSecond устанавливается значение «Использовать уровни», чтобы привести их в соответствие со структурой входных данных.
