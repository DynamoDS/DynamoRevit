## Подробности
Этот узел служит для создания нового элемента разделенной поверхности на выбранной грани элемента Revit и определяет его компоновку с использованием заданных делений U и V и, при необходимости, угол поворота.
Разделенная поверхность представляет собой сетку с образцом, применяемую к грани, которая обычно используется для размещения панелей витража, адаптивных компонентов или панельных систем на поверхности формы.

Деления U и V определяют количество делений в каждом направлении поверхности, а параметр поворота регулирует ориентацию сетки относительно системы координат U-V.

В этом примере выбрана грань, которая используется в качестве входных данных для поверхности для узла DividedSurface.ByFaceUVDivisionsAndRotation, а также UDivs, VDivs и gridRotation, для управления которыми используются регуляторы. В последних узлах отображаются значения разделенной поверхности. При запуске этого примера графика необходимо просмотреть предупреждение Revit и удалить предлагаемые элементы, чтобы сетки появились на выбранной поверхности.
