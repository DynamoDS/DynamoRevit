## Informacje szczegółowe
`Tag.LeaderElbow` returns the bend point of a tag’s leader line. The input is a Revit tag element, and the output is a Dynamo point representing the leader elbow location.

In the example below, a tag is selected from the active Revit view and used as the input for `Tag.LeaderElbow`. The output is the point location where each tag leader bends.
