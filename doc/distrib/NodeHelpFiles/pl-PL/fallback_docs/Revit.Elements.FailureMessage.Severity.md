## Informacje szczegółowe
Ten węzeł pobiera poziom istotności danego obiektu FailureMessage. Wartości ważności to: „Warning”, „Error” i „Critical”.

W tym przykładzie wykonywana jest reguła doradcy wydajności w celu sprawdzenia warunku „przycinanie widoku jest wyłączone”. Wynikowe komunikaty o błędach są przekazywane do węzła FailureMessage.Severity, w którym zwracany jest poziom istotności każdego problemu.
