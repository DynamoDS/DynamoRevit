## Informacje szczegółowe
Ten węzeł ocenia podany element i zwraca wartość logiczną (Boolean) wskazującą, czy element ma zdefiniowane położenie, wraz z komunikatami wyjątków wyjaśniającymi, dlaczego może go nie mieć.

W tym przykładzie element jest definiowany przez identyfikator elementu przy użyciu węzła Element By Id, a następnie przekazywany do węzła Element.HasActualLocation. Dane wyjściowe wskazują wartość true (prawda), jeśli element ma rzeczywiste położenie, lub false (fałsz) z komunikatem wyjątku, jeśli go nie ma.
