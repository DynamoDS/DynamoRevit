## Informacje szczegółowe
Węzeł `Element.UnjoinGeometry` rozłącza geometrię dwóch danych elementów.

W poniższym przykładzie w bieżącym dokumencie programu Revit zostaje wybrany strop. Połączone elementy zostają zebrane, a następnie odłączone od stropu wejściowego.
