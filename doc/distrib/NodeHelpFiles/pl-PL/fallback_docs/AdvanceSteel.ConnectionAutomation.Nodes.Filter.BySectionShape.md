## Informacje szczegółowe
Ten węzeł zwraca listę węzłów połączeń filtrowanych według wybranego kształtu przekroju.

W tym przykładzie po wybraniu serii elementów danych konstrukcyjnych z wieloma węzłami połączeń lista węzłów połączeń jest filtrowana. Pozostają te, które pasują do wybranego kształtu przekroju. Elementy danych konstrukcyjnych mogą mieć listy podrzędne elementów, dlatego indeks jest ustawiony na 0 — wartość domyślną.

W tym przykładzie wybrano grupę równoległych kołnierzy dwuteowych i profil zamknięty, a wynik składa się z listy słowników z właściwościami „accepted” (zaakceptowane) i „rejected” (odrzucone). Akceptowaną wartością jest węzeł połączenia filtrowany według wybranego kształtu przekroju. W tym przypadku kształt przekroju to profil zamknięty.
