## Informacje szczegółowe
Ten węzeł zwraca kształt przekroju elementu danych konstrukcyjnych.

W tym przykładzie wybranie elementu danych belki/słupa konstrukcyjnego powoduje zwrócenie kształtu przekroju konstrukcyjnego. W tym przypadku jest to „I-shape Parallel Flange” (Dwuteownik równoległościenny).
