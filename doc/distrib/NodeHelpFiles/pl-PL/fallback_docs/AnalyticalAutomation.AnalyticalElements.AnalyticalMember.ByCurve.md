## Informacje szczegółowe
Ten węzeł tworzy liniowy element analityczny na podstawie krzywej.

W tym przykładzie definiowane są dwa punkty przez wyodrębnienie położeń przecięć linii siatki w położeniach A4 i A5 w modelu konstrukcyjnym Snowdon. Poziom L2 TOS służy do ustawiania położenia pionowego obu punktów. Zostaje utworzona linia przez połączenie tych punktów i przekazana jako dane wejściowe tego węzła. Dane wyjściowe to liniowy element analityczny reprezentujący zdefiniowaną krzywą.
