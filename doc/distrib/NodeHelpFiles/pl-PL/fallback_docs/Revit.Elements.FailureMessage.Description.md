## Informacje szczegółowe
Ten węzeł pobiera szczegółowy opis błędu lub ostrzeżenia wygenerowanego przez regułę doradcy wydajności. Tłumaczy błąd na jasny i czytelny tekst, dzięki czemu użytkownicy mogą lepiej zrozumieć problem.

W tym przykładzie reguła doradcy wydajności sprawdza, czy „przycinanie widoku jest wyłączone”. Wyniki są przetwarzane za pośrednictwem węzła FailureMessage.Description, który zwraca opisowy komunikat wyjaśniający przyczynę błędu.
