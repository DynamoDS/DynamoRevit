## Informacje szczegółowe
Ten węzeł pobiera rodzinę i typ rodziny wybranego komponentu adaptacyjnego.

W tym przykładzie wszystkie umieszczone komponenty adaptacyjne są następnie przekazywane jako dane wejściowe do węzła AdaptiveComponent.Type. Dane wyjściowe to nazwy rodzin i typów rodzin.

Aby można było uruchomić ten przykładowy plik pomocy dla węzłów, należy wczytać plik „Diagnostics Tripod-1 point.rfa” do pliku programu Revit. Rodzina jest przechowywana tutaj: C:\ProgramData\Autodesk\RVT 2027\Dynamo\Samples\Data
