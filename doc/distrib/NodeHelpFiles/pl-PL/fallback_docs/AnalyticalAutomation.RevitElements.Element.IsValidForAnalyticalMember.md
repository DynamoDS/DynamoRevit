## Informacje szczegółowe
Ten węzeł ocenia wybrany element programu Revit, aby określić, czy jest odpowiedni do użycia jako liniowy element analityczny. Jeśli element jest nieprawidłowy, wynik zawiera komunikat o wyjątku z wyjaśnieniem przyczyny.

W tym przykładzie element jest definiowany przez identyfikator elementu i przekazywany do węzła Element.IsValidForAnalyticalMember. Dane wyjściowe zawierają wynik logiczny (Boolean) wskazujący poprawność elementu i komunikat o wyjątku, jeśli ma to zastosowanie.
