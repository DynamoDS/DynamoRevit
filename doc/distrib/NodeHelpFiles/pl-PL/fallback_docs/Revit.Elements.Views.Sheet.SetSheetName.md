## Informacje szczegółowe
Węzeł `Sheet.SetSheetName` ustawia dla danego arkusza wejściową nazwę arkusza (string).

W poniższym przykładzie do wszystkich nazw arkuszy w modelu dodawany jest przedrostek „(E)”.
