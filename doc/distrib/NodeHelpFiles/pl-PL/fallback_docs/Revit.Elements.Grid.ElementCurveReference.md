## Informacje szczegółowe
Ten węzeł zwraca odniesienie krzywej programu Revit skojarzone z danym elementem siatki.

W tym przykładzie zostają wybrane wszystkie elementy siatki w projekcie i zostają one użyte jako dane wejściowe węzła GridElementCUrveReference. Ten węzeł udostępnia to odniesienie, dzięki czemu można go używać w innych operacjach, takich jak wymiarowanie, umieszczanie elementów na podstawie ścieżki lub analiza geometryczna.
